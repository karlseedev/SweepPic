## 7. 디스크 캐시 정책 확정 (2025-12-23)

### 최종 정책

**"디스크 캐시는 초기 프리로드 전용"**

| 용도 | 디스크 캐시 사용 | 설명 |
|------|----------------|------|
| 콜드 런치 초기 프리로드 | ✅ 사용 | `startInitialPreload()` → 디스크 → 메모리 |
| 스크롤 중 셀 구성 | ❌ 스킵 | 메모리 → Pipeline 직접 연결 |

### 근거

1. **디스크 캐시 ON**: 회색 썸네일 아주 많음 (I/O 지연)
2. **디스크 캐시 OFF**: 회색 썸네일 거의 안 보임 (Pipeline degraded first-paint가 더 빠름)
3. **초기 프리로드**: 디스크 캐시 사용해도 문제 없음 (100ms 내 완료)

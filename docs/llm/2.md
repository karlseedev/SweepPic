# 저품질 사진 판별 로직 기획 (요약)

## 결론 요약
- 현재 기획은 **룰 기반(on-device, no-training)에서 최선에 가까운 구조**다.
- 다만 임계값은 **기기/촬영환경/데이터셋 의존**이므로, “최선”을 증명하려면 **캘리브레이션**이 필요하다.

## 핵심 이유
- 외부 자료는 “무엇이 저품질인가”를 정의하지만, **절대 임계값**을 보장하지 않는다.
- 따라서 실제 사진 샘플을 통해 임계값을 **데이터 기반으로 확정**해야 정확도/재현율 균형이 완성된다.

## 향후 발전 방향 (선택지)
1) **현 기준 유지 + 캘리브레이션 절차 추가**
   - 샘플 라벨링 → 지표 분포 분석 → 임계값 최적화
2) **외부 자료 기반 대안 지표/모델 비교**
   - NIQE/BRISQUE, Vision Aesthetics 등 검토

## 권장 다음 단계
- 정확도를 보장하려면 **캘리브레이션 계획 문서화**가 우선.
- 필요 시 웹조사로 대안 지표까지 확장 검토.

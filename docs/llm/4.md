### ⚠️ TrashDataSource 인덱스 캐시 미적용

계획 문서에는 O(1) 조회를 위한 인덱스 캐시가 명시됨:
```swift
// 계획
private var indexCache: [String: Int] = [:]
func assetIndex(for assetID: String) -> Int? {
    return indexCache[assetID]  // O(1)
}
```

**현재 구현**:
```swift
// 실제 (O(N) 선형 탐색)
func assetIndex(for assetID: String) -> Int? {
    assets.firstIndex { $0.localIdentifier == assetID }
}
```

**영향**: 휴지통 사진 수가 많을 때 성능 저하 가능

## 1차 출시 이후 추가 예정 기능

### A. 판별 모드 옵션

정리 실행 전 사용자가 선택할 수 있는 옵션 제공.

| 옵션 | 설명 | 기준 |
|-----|------|------|
| **신중한 정리** (기본) | 오탐지 최소화 | Precision 모드 (Strong 신호만) |
| **적극적 정리** | 커버리지 최대화 | Recall 모드 (Strong + Conditional + Weak) |

**Recall 모드 상세**:
- Strong: 극단 노출, 심각 블러
- Conditional: 주머니 샷, 극단 단색, 렌즈 가림
- Weak: 일반 블러(2점), 일반 노출, 낮은 색상 다양성, 저해상도(각 1점)
- Weak 합산 >= 3점 → 저품질 판정

---

### B. 스크린샷 포함 옵션

정리 대상에 스크린샷을 포함할지 선택 가능.

| 옵션 | 설명 |
|-----|------|
| **스크린샷 제외** (기본) | 스크린샷은 정리 대상에서 제외 |
| **스크린샷 포함** | 스크린샷도 저품질 기준으로 분석하여 정리 |

**스크린샷 저품질 기준** (별도 정의 필요):
- 중복 스크린샷 감지
- 오래된 스크린샷 (N개월 이상)

## UI 설계 (예정)

### 옵션 선택 방식

정리 방식 선택 팝업에 옵션 추가 또는 설정 화면에서 토글 제공.

```
[정리 방식 선택]
├─ 최신사진부터 정리
├─ 이어서 정리
├─ 연도별 정리
│
├─ ⚙️ 옵션
│   ├─ [ ] 적극적 정리 (더 많은 사진 정리)
│   └─ [ ] 스크린샷 포함
```

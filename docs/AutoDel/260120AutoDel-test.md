## 1. 테스트 데이터셋

## 2. iCloud 썸네일 캐시 검증 (※필수)

| 검증 항목 | 테스트 방법 | 예상 결과 |
|----------|------------|----------|
| **썸네일 캐시 크기** | "Optimize Storage" 모드에서 256×256 요청 | ~342×256 로컬 캐시 반환 |
| **네트워크 미사용** | `networkAccessAllowed=false`로 요청 | 원본 다운로드 없이 성공 |
| **분석 정확도 (노출)** | 64×64 썸네일 vs 원본으로 노출 분석 비교 | 결과 일치 |
| **분석 정확도 (블러)** | 256×256 썸네일 vs 원본으로 블러 분석 비교 | 결과 일치 |
| **다양한 iCloud 상태** | 로컬만/혼합/전체 iCloud 환경 테스트 | 모든 환경에서 분석 가능 |

> **실패 시 대응**: 썸네일 캐시가 불충분한 경우, 원본 다운로드 옵션 복원 고려

---

## 3. 임계값 검증 (※필수)

| 검증 항목 | 테스트 방법 | 조정 기준 |
|----------|------------|----------|
| **Laplacian 50/100** | 심각 블러/일반 블러 샘플로 감지율 측정 | 오탐/미탐 비율 |
| **휘도 0.10/0.90** | 극단 노출 샘플로 감지율 측정 | 야경/실루엣 오탐 여부 |
| **RGB Std 10/15** | 단색/다채로운 이미지 분포 측정 | 문서/메모 사진 오탐 여부 |
| **Face Quality 0.4** | 흐릿한 얼굴/선명한 얼굴 샘플로 측정 | 블러 Safe Guard 적절성 |
| **AestheticsScore -0.3/0** | iOS 18 기기에서 다양한 품질 이미지 테스트 | Precision/Recall 목표 달성 |
| **비네팅 0.05** | 주머니 샷/정상 사진 감지율 측정 | 비네팅 사진 오탐 여부 |

## 4. 목표 지표

| 모드 | Precision | Recall |
|-----|:---------:|:------:|
| Precision | >= 95% | >= 60% |
| Recall | >= 80% | >= 85% |

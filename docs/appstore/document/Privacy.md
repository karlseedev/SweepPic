# SweepPic 개인정보 처리방침

최종 수정일: 2026년 3월 11일
시행일: 2026년 3월 11일

## 1. 개요

SweepPic(이하 "앱")는 사진 정리에 특화된 iOS 앱입니다.
사진 데이터와 얼굴 인식 결과는 기기 내에서만 처리되며,
외부 서버로 전송되지 않습니다.

## 2. 수집하는 개인정보 항목 및 수집 방법

앱은 서비스 개선을 위해 다음 데이터를 자동으로 수집합니다:

**a) TelemetryDeck (분석)**

TelemetryDeck은 이중 해싱 등 식별자 보호 조치를 적용하여,
원본 식별자가 그대로 저장되지 않도록 처리합니다.

| 항목 | 예시 |
|------|------|
| 앱 사용 이벤트 | 열람·삭제·분석 횟수 |
| 사진 규모 범위 | "1K-5K" (개수 구간만) |

※ IDFV는 SDK 내부에서 해싱 처리되어 원본 식별자가 보존되지 않습니다.

**b) Supabase (자체 분석 서버)**

| 항목 | 예시 |
|------|------|
| 앱 사용 이벤트 | 열람·삭제·분석 횟수 |
| 기기 식별자 (IDFV) | 기기별 고유값 |
| 기기 정보 | iPhone16,1 / iOS 18.3 / 앱 v1.0 |
| 구독 상태 | free 또는 pro |
| 사진 규모 범위 | "1K-5K" (개수 구간만, 사진 내용 아님) |
| 추천 프로그램 사용자 ID | Keychain에 저장되는 앱 내부 UUID |
| 초대 코드 및 초대 상태 | 초대 코드 입력, 적용 여부, 보상 상태 |
| 보상 처리 정보 | 보상 ID, 상품 ID, 구독 상태, StoreKit 거래 ID |
| 푸시 알림 토큰 | APNs device token |

※ 사진 파일, 사진 메타데이터, 얼굴 데이터는 수집하지 않습니다.

## 3. 개인정보 처리 목적

- 앱 기능 개선 및 오류 진단
- 사용 패턴 분석을 통한 서비스 품질 향상
- 광고 표시 (Google AdMob)
- 구독 상태 확인 및 유료 기능 제공
- 초대 코드 적용, 추천 보상 지급 및 부정 사용 방지
- 추천 보상 및 앱 관련 알림 발송

## 4. 개인정보 보유 및 이용 기간

- TelemetryDeck 분석 데이터: TelemetryDeck의 데이터 보관 정책에 따름
- Supabase 분석 이벤트: 수집일로부터 365일 보관 후 자동 파기
- 추천 프로그램 사용자 ID 및 추천/보상 처리 기록: 추천 기능 제공, 보상 처리, 부정 사용 방지 및 고객 지원에 필요한 기간 동안 보관
- 푸시 알림 토큰: 알림 제공에 필요한 기간 동안 보관, 권한 철회 또는 토큰 갱신 시 갱신·삭제
- 앱 삭제 시 기기 내 앱 데이터 삭제. 단, Keychain에 저장된 추천 프로그램 사용자 ID는 iOS 정책에 따라 앱 재설치 후에도 유지될 수 있음

## 5. 개인정보 처리위탁 및 국외 이전

앱은 서비스 제공을 위해 다음 외부 서비스를 이용합니다.

| 수탁자/제공자 | 처리 항목 | 목적 |
|--------------|----------|------|
| Apple Inc. | 구독 구매, 결제, 환불 및 거래 정보 | 구독 결제 및 구매 복원 |
| TelemetryDeck GmbH | 앱 사용 이벤트, 분석 식별자 | 사용 분석 |
| Supabase Inc. | 사용 이벤트, IDFV, 기기 정보, 구독 상태, 추천 프로그램 사용자 ID, 초대/보상 처리 정보, 푸시 알림 토큰 | 분석, 추천/보상 기능 제공, 알림 |
| Google LLC | 광고 표시를 위한 기기·네트워크 정보 및 광고 관련 식별자 | 광고 제공 |

※ Google AdMob SDK는 광고 표시를 위해 기기 및 네트워크 정보를
독립적으로 수집할 수 있습니다. ATT(앱 추적 투명성) 동의 없이는
맞춤 광고를 제공하지 않습니다.

## 7. 얼굴 인식 기술 안내

앱은 Apple Vision Framework를 사용하여 사진 내 얼굴을
자동으로 감지하고 확대하는 기능을 제공합니다.

- 모든 처리는 기기 내에서만 수행됩니다
- 얼굴 데이터는 저장되거나 외부로 전송되지 않습니다
- 마케팅, 광고, 데이터마이닝에 사용하지 않습니다

## 8. 자동화된 의사결정 안내

앱의 "자동 정리" 기능은 사진 품질(노출, 블러 등)을
AI로 분석하여 삭제 대상을 제안합니다.

- 자동 삭제는 수행하지 않으며, 최종 결정은 사용자가 합니다
- 분석 결과는 기기 내에서만 처리됩니다

## 9. 행태정보 수집·이용·거부 안내

앱은 자체적으로 광고 개인화를 위한 행태정보를 수집하지 않습니다.
다만 Google AdMob을 통해 광고가 표시되며, Google SDK가 광고 제공 및
측정을 위해 관련 정보를 처리할 수 있습니다.
ATT 동의 없이는 앱 간 추적 기반의 개인화 광고를 제공하지 않습니다.

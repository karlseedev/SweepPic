### Description (설명) — 4,000자 이내

```
사진이 수만 장 쌓여 있는데, 정리할 엄두가 안 나시나요?

AI 자동 삭제는 불안하고, 한 장씩 고르기엔 시간이 너무 걸리죠. SweepPic으로 사진 앱처럼 보면서 빠르게 정리하세요.

[스와이프 삭제 - 보면서 바로 정리]
사진을 스크롤하다가 필요 없는 사진을 가로로 스와이프하면 바로 삭제대기함으로 이동합니다. 확인 팝업 없이 빠르게 정리하세요. 실수로 지워도 삭제대기함에서 언제든 복구할 수 있습니다.

[얼굴 크롭 비교 - 표정까지 한눈에]
연속 촬영한 사진에서 누가 눈을 감았는지, 어떤 표정이 더 좋은지 궁금하셨죠? SweepPic은 유사한 사진에서 동일 인물의 얼굴을 자동으로 크롭하고 확대하여 나란히 보여줍니다. 축소된 썸네일이 아닌, 확대된 얼굴로 미세한 차이를 직관적으로 비교하세요.

[AI 자동 정리 - 저품질 사진 자동 감지]
흔들린 사진, 너무 어둡거나 밝은 사진을 AI가 자동으로 찾아냅니다. 3단계(완화/기본/강화)로 정리 범위를 직접 조절할 수 있어, 내 기준에 맞는 정리가 가능합니다. 즐겨찾기, 편집한 사진, 인물 사진은 자동으로 보호됩니다.

[삭제대기함 - 안전한 2단계 삭제]
삭제한 사진은 바로 사라지지 않고 삭제대기함에 보관됩니다. 마음이 바뀌면 언제든 복구하세요. "비우기"를 누르면 iOS 시스템 휴지통으로 이동하여 30일간 추가 복구 기회가 있습니다.

[완전한 프라이버시]
모든 사진 분석과 얼굴 인식은 기기에서만 수행됩니다. 사진 이미지는 외부 서버로 전송되지 않습니다. 앱 개선을 위한 익명 사용 통계만 수집됩니다.
```

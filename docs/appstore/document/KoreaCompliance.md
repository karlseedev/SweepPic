---

## 1. App Store Connect 입력

**경로:**
```
App Store Connect > 앱 선택 > General > App Information > Korean Law
```

| 항목 | 입력값 | 비고 |
|------|-------|------|
| 이메일 | karlseee4@gmail.com | 필수 |
| 사업자등록번호 (BRN) | 000-00-00000 | 사업자등록 후 입력 |

> Individual 계정: 이메일 + BRN 필수
> Organization 계정: + 전화번호 + 회사명 추가 필수

## 2. 사전 준비 (출시 전 필수)

### 2-1. 사업자등록

| 항목 | 내용 |
|------|------|
| 신청 | 홈택스 (즉일~1일) |
| 업종코드 | 722000 (소프트웨어 개발) + 525101 (통신판매업) |
| 사업장 | 자택 가능 (임대차계약서 불필요) |
| 비용 | 무료 |

> 2024.12부터 App Store에 사업자등록번호 표시 의무화

### 2-2. 통신판매업 신고

| 항목 | 내용 |
|------|------|
| 신청 | 정부24 (3~6일) |
| 필요 서류 | 사업자등록증 사본 + 구매안전서비스 비적용 확인서 |
| 취급품목 | 모바일 애플리케이션 |
| 비용 | 등록면허세 약 40,500원 (매년 1월 갱신) |

> 인앱 자동갱신 구독 = 전자상거래 → 통신판매업 신고 필수

### 미등록 리스크

| 항목 | 리스크 |
|------|-------|
| 사업자등록 미등록 | 매출 1% 가산세, 매입세액 불공제 |
| 통신판매업 미신고 | 영업정지 15일+, 최고 500만원 벌금 |

## 3. 앱 내 사업자정보 표시 (전자상거래법 제10조)

인앱 결제가 있는 앱은 사업자 정보를 앱 내에서 확인할 수 있어야 합니다.

**필수 표시 항목:**

| 항목 | 예시 |
|------|------|
| 상호 | (사업자등록 후 확정) |
| 대표자 | Simon Karl |
| 사업자등록번호 | 000-00-00000 |
| 통신판매업 신고번호 | 제0000-서울OO-0000호 |
| 주소 | (사업장 주소) |
| 이메일 | karlseee4@gmail.com |

**구현 방법:** 설정 화면에 "사업자 정보" 섹션 추가, 또는 지원 페이지 링크로 대체 가능

> 위반 시 과태료: 1차 100만원, 2차 200만원, 3차 500만원

## 5. 작업 순서 요약

```
1. 사업자등록 (홈택스) → BRN 발급
2. 통신판매업 신고 (정부24) → 신고번호 발급
3. App Store Connect Korean Law 입력 (이메일 + BRN)
4. 앱 내 사업자정보 표시 구현 (설정 화면 또는 지원 페이지)
```

# Part 1. 전체 기능 명세

---

## 1. 그리드 뷰 (메인 사진 브라우징)

**위치**: `SweepPic/Features/Grid/`

- 3열 정사각형 그리드 (고정)
- ImagePipeline 기반 비동기 썸네일 로딩 + 프리히트(±1행 미리 로드)
- 스크롤 중 저화질 썸네일로 전환하여 120fps 유지 (5만 장 이상 지원)
- 삭제대기함에 있는 사진은 50% 검정 오버레이로 어둡게 표시 (딤드)
- 유사사진이 감지된 사진에는 테두리 표시

### 선택 모드
- 탭으로 개별 선택, 드래그로 연속 다중 선택
- 선택한 사진들 일괄 삭제 가능

---

## 2. 뷰어 (전체화면 사진 보기)

**위치**: `SweepPic/Features/Viewer/`

- UIPageViewController 기반 좌우 스와이프 탐색
- 더블탭 2배 줌, 핀치 자유 줌 (1~3배)
- 양옆 사진 미리 로드

### 모드별 동작
| 모드 | 진입 경로 | 주요 버튼 |
|------|----------|----------|
| 일반(normal) | 메인 그리드에서 셀 탭 | 삭제(빨간), 복구(초록, 스와이프 복구 사진만) |
| 삭제대기함(trash) | 삭제대기함에서 셀 탭 | 복구(초록), 최종 삭제(빨간) |
| 정리 미리보기(cleanup) | 자동정리 세션 중 | 제외 버튼 (정리 대상에서 빼기) |

### 유사사진 (+) 버튼
- 유사사진이 감지된 사진에만 하단 우측에 표시
- 탭 시 얼굴 비교 화면(FaceComparisonViewController)으로 이동

---

## 3. 스와이프 삭제

- 그리드에서 썸네일을 보면서 **가로 스와이프**하면 확인 팝업 없이 즉시 삭제(삭제대기함으로 이동)
- 삭제 후 자동으로 다음 사진으로 이동
- 삭제대기함이 안전장치이므로 확인 절차 없이 빠른 삭제 가능

---

## 4. 유사사진 분석 및 얼굴 비교

**위치**: `SweepPic/Features/SimilarPhoto/`

### 4-1. 유사사진 분석 (SimilarityAnalysisQueue)
- Vision Framework의 Feature Print 기반 이미지 유사도 비교
- 연속 촬영된 유사 사진을 자동으로 그룹화 (최소 3장 이상)
- FIFO 큐로 분석 요청 처리 (동시 5개, 과열 시 2개)
- 캐시 hit 시 100ms 이내, miss 시 0.5초 이내 결과 반환

### 4-2. 얼굴 감지 및 크롭 (FaceDetector, FaceCropper)
- Vision의 VNDetectFaceRectanglesRequest로 얼굴 감지
- 화면 너비 5% 미만 얼굴 제외, 크기순 상위 5개만 분석
- 감지된 얼굴을 바운딩박스 기준으로 자동 크롭
- VNDetectFaceCaptureQualityRequest로 얼굴 품질 평가

### 4-3. 얼굴 인식 및 매칭 (SFaceRecognizer)
- 정렬된 얼굴을 페이스 임베딩(특징 벡터)으로 변환
- 코사인 유사도로 동일 인물 식별
- 동일 인물끼리 그룹화하여 인물별 페이지로 제공

### 4-4. 얼굴 비교 화면 (FaceComparisonViewController)
- **2열 정사각형 그리드**에 크롭된 얼굴 이미지 표시 (최대 8장)
- 헤더: "인물 N (M장)" + 순환 버튼 (다음 인물 페이지)
- 각 얼굴 아래 "사진 번호 / 전체 개수" 표시 (예: "3/12")
- 현재 보고 있던 사진에 초록색 테두리
- 얼굴 탭으로 선택/해제, 선택한 사진 일괄 삭제대기함 이동
- 좌우 스와이프로 인물 간 전환 (UIPageViewController)

**핵심 가치**: 축소된 썸네일이 아닌, 크롭·확대된 얼굴로 표정·눈감김 등 미세한 차이를 직관적으로 비교 가능

---

## 5. 자동정리 (AutoCleanup)

**위치**: `SweepPic/Features/AutoCleanup/`

### 5-1. 품질 분석 파이프라인 (4단계)

**Stage 1 — 메타데이터 필터**
- 즐겨찾기/편집됨/숨김/공유앨범 → SKIP (Safe Guard)
- 스크린샷/iCloud 전용/5초 초과 영상 → SKIP

**Stage 2 — 노출 분석 (ExposureAnalyzer)**
- 극단 어두움(휘도 < 0.10) / 극단 밝음(휘도 > 0.90) → Strong 신호
- 일반 어두움(< 0.15) / 일반 밝음(> 0.85) → Weak 신호

**Stage 3 — 블러 분석 (BlurAnalyzer)**
- Metal(Laplacian Variance) 기반 흐림 감지
- 심각 블러(< 50) → Strong, 경미 블러(< 200) → Weak
- iOS 18+: AestheticsScore 우선 사용

**Stage 4 — Safe Guard (SafeGuardChecker)**
- Portrait 모드(심도 효과) → 블러 판정 무효화
- 선명한 얼굴(품질 >= 0.4) → 블러 판정 무효화

### 5-2. 판별 단계 (점수 기반 3단계)
| 단계 | 점수 범위 | 대상 |
|------|:--------:|------|
| 완화 (light) | 30점 이하 | 확실한 저품질 (극단 블러, 극단 노출) |
| 기본 (standard) | 31~40점 | 중간 저품질 추가 |
| 강화 (deep) | 41~50점 | 경미한 저품질까지 포함 |

- 기본적으로 완화 단계만 표시, "더 보기/덜 보기" 버튼으로 단계 확장/축소

### 5-3. 정리 세션 (CleanupSession)
- 정리 방식: 최신부터(fromLatest) / 이어서(continueFromLast) / 연도별(byYear)
- 최대 2,000장 검색, 최대 50장 저품질 탐지 시 중단
- 탐지된 사진을 미리보기 화면에서 확인 → 제외 가능 → 일괄 삭제대기함 이동
- "이어서 정리" 지원 (마지막 탐색 위치 JSON 저장)

### 5-4. 동영상 처리
- 1초 미만: 저품질 확정 (실수 촬영)
- 1~5초: 3개 프레임 추출하여 분석
- 5초 초과: SKIP

# Part 2. 특화 기능 (경쟁 차별점)

> 기존 사진 앱/정리 앱에 없거나 부족한, SweepPic만의 핵심 차별 기능

---

## 1. 목록에서 밀어서 삭제 (썸네일 보면서 즉시 삭제)

**무엇이 다른가**: 기존 앱은 사진 삭제 시 "선택 → 삭제 버튼 → 확인 팝업" 3단계를 거침. SweepPic는 그리드에서 썸네일을 보면서 **가로 스와이프 한 번**이면 바로 삭제. 사진을 훑어보다가 불필요한 사진이 보이면 그 자리에서 바로 지울 수 있음.

**동작 흐름**:
1. 그리드에서 썸네일을 보며 사진을 훑어봄
2. 불필요한 사진 발견 → 가로 스와이프 (화면 20% 이상 또는 빠른 속도)
3. 햅틱 피드백으로 삭제 인지
4. 확인 팝업 없이 즉시 삭제대기함으로 이동
**왜 가능한가**: 삭제대기함이라는 안전장치가 있어서 확인 팝업이 필요 없음. "삭제 = 즉시 영구 삭제"가 아니라 "삭제 = 삭제대기함으로 이동"이므로 언제든 복구 가능.

---

## 2. 유사사진 얼굴 크롭·확대 비교

**무엇이 다른가**: 기존 앱은 유사 사진을 보여줄 때 축소된 썸네일 그대로 나열함. SweepPic는 유사 사진 그룹에서 **동일 인물의 얼굴만 자동으로 크롭·확대**하여 표정 차이를 한눈에 비교 가능.

**동작 흐름**:
1. 뷰어에서 유사사진이 감지된 사진에 (+) 버튼 표시
2. (+) 탭 → 얼굴 비교 화면 진입
3. 동일 인물의 얼굴이 크롭·확대되어 2열 그리드로 나열 (흰색 테두리)
4. 인물이 여러 명이면 좌우 스와이프로 인물별 페이지 전환
5. 얼굴 탭으로 선택 → 불필요한 사진 일괄 삭제

**기술 구성**:
- Vision Framework로 얼굴 감지 + 품질 평가
- 페이스 임베딩(특징 벡터) + 코사인 유사도로 동일 인물 식별
- 자동 크롭: 얼굴 바운딩박스 기준 정사각형 크롭

**핵심 가치**: 연속 촬영한 단체사진에서 "누가 눈 감았는지, 누구 표정이 더 좋은지"를 썸네일이 아닌 확대된 얼굴로 직관적 비교.

---

## 3. 자동정리 (점수 기반 3단계 선택 삭제)

**무엇이 다른가**: 저품질 사진 자동감지 자체는 기존 앱과 유사함. SweepPic의 차별점은 **점수 기반 3단계(완화/기본/강화)로 저품질을 분류**하고, 사용자가 "더 보기/덜 보기" 버튼으로 정리 범위를 직접 조절할 수 있다는 점.

**3단계 구조**:
| 단계 | 점수 범위 | 대상 |
|------|:--------:|------|
| 완화 (light) | 30점 이하 | 확실한 저품질 (극단 블러, 극단 노출) |
| 기본 (standard) | 31~40점 | 중간 저품질 추가 |
| 강화 (deep) | 41~50점 | 경미한 저품질까지 포함 |

- 기본적으로 완화 단계만 표시, "31~40점 사진 N장 더 보기" 버튼으로 기본 단계 확장
- 기본 단계에서 "41~50점 사진 N장 더 보기"로 강화 단계까지 확장 가능
- 반대로 "N장 덜 보기"로 단계 축소도 가능
- Safe Guard로 즐겨찾기/편집됨/Portrait 모드 등 중요 사진 자동 보호
- 탐지된 사진은 미리보기에서 확인 후 최종 결정

**핵심 가치**: "얼마나 깐깐하게 정리할지"를 사용자가 점수 단계로 직접 조절.

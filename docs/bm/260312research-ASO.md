## 3. ASO 관련 디자인 비용

### 앱스토어 스크린샷 (5~10장 기준)

| 방법 | 비용 |
|------|------|
| DIY 도구 (Figma, Canva) | 무료 |
| AppLaunchpad | 무료~$24/월 (템플릿 기반, 자동 리사이징) |
| AppScreens | $75/년 ($24/월, AI 캡션 생성) |
| Fiverr 프리랜서 (저가) | $5~$50 |
| Fiverr 프리랜서 (중급) | $50~$200 (커스텀 디자인) |
| Upwork 프리랜서 | $100~$500 (고품질) |
| 전문 에이전시 | $500~$2,000+ (A/B 테스트 포함) |

### App Preview 영상 (15~30초)

| 방법 | 비용 |
|------|------|
| DIY (iPhone 화면 녹화 + iMovie) | 무료 |
| 프리랜서 (기본, 화면 녹화 기반) | $400~$800 |
| 프리랜서 (중급, 모션 그래픽 포함) | $600~$1,500 |
| 에이전시 (풀 프로덕션) | $2,000~$5,000+ |

|------|------|

| SplitMetrics Optimize | $13,600/년~ (1앱, 1로케일, 12테스트/년) |

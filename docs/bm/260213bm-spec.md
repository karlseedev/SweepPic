## 1. 제품 철학 & 수익화 원칙 ✅

### 구조

```
[사진 정리]        →  [휴지통]       →  [비우기]
항상 무료, 무제한      무료 보관         여기만 게이트
```

- **무료 영역**: 스와이프 삭제, 유사 사진 분석, 자동 정리, 복구 — 전부 무제한
- **게이트**: "비우기"(영구 삭제) 한 곳에만 존재

### 광고 배치 방향

- 정리 행위 **도중**에는 광고 없음 (비교 화면, 뷰어, 그리드)
- 정리 행위 **완료 시점**에 전면 광고 (짝수 회차만, 스킵 가능)
- **대기 화면**에 배너 (분석 중 하단)

| 시점 | 타입 | 빈도 |
|------|------|------|
| 휴지통 비우기 (한도 초과) | 리워드 | 일 3회 |
| 유사사진 삭제 완료 | 전면 | 짝수 회차 |
| 자동정리 휴지통 이동 완료 | 전면 | 짝수 회차 |
| 사진 분석 대기 화면 | 배너 | 매회 |

### 운영 방향

- 사진 정리 행위는 게이트로 막지 않는다 (비우기만 게이트)
- 그리드, 뷰어, 비교 화면 등 탐색 중에는 광고를 넣지 않는다
- 휴지통 복구는 유료화하지 않는다
- Plus 구독자에게는 광고를 보여주지 않는다
- 기존 무료 기능을 나중에 유료로 전환하지 않는다

## 4. Free Tier 설계 ✅

### 일일 한도

- **일일 10장** 무료 영구삭제 (방법 불문)
- 매일 자정 리셋
- 숫자는 추후 A/B 테스트로 조정 가능
- 자동 정리 실행 횟수: **무제한** (게이트는 비우기에만)
- 한도 초과 시 관용 없음 — 12장이어도 무료 비우기 불가, 게이트 표시

### 게이트 판단

```
비우기 탭
  ↓
휴지통 ≤ 남은 무료 → 게이트 없이 바로 iOS 팝업 → 삭제
휴지통 > 남은 무료 → 게이트 시트 표시
```

### Grace Period

- 설치 후 3일: 광고 없음, 게이트 없음, 카운터 없음 → 무제한
- Day 3 마지막 비우기 후: 종료 예고 알림
- Day 4 첫 게이트: 1회 무료 비우기 쿠폰 (소프트 랜딩)
- Day 5부터: 정상 운영

## 5. 광고 모델 📝

| 광고 | 트리거 | 빈도 | Phase |
|-----|--------|------|-------|
| **리워드** | 사용자가 게이트에서 선택 | 일 3회 | Phase 1 |
| **전면** | 비우기 완료 후 자동 | 세션 1회 | Phase 3 |
| **배너** | 자동 정리 결과 화면 | 결과 화면마다 | Phase 1 |

### 리워드 광고 보상
- 1회 시청 = 현재 휴지통 전체 비우기 (최대 50장)
- 50장 초과 시 오래된 순 50장만 삭제, 나머지 남음
- 무료 카운터는 소모 안 함 (광고가 대신 지불)
- 3회 소진 시 → 골든 모먼트 (Plus 전환 유도)

### 리워드 시청 후 iOS 팝업 취소 대응
- 5분 유예 토큰: 5분 이내 재시도 시 추가 광고 없이 삭제 허용

### 광고 금지 구역
- 사진 보기/탐색 (그리드, 뷰어)
- 스와이프 삭제 중
- 유사 사진 비교 중
- Grace Period (3일)
- Plus 구독자

## 8. 게이트 & 카운터 UI 📝

### 게이트 시트 (남은 무료 > 0)

```
┌────────────────────────────────────────┐
│  무료 삭제가 N장 남았어요                 │
│  휴지통: M장                            │
│                                        │
│  [N장만 삭제 (무료)]                     │
│  [광고 보고 전체 비우기]                  │
│  [Plus — 무제한]                        │
│  [나중에]                               │
└────────────────────────────────────────┘
```

### 게이트 시트 (남은 무료 = 0)

```
┌────────────────────────────────────────┐
│  오늘의 무료 삭제를 모두 사용했어요        │
│                                        │
│  [광고 보고 비우기] (N/3)                │
│  [Plus — 무제한]                        │
│  [내일 다시 올게요]                       │
└────────────────────────────────────────┘
```

### 카운터 (듀오링고식)

사용자가 남은 무료 삭제 수를 항상 볼 수 있게 표시:

- **휴지통 탭 아이콘**: 뱃지로 남은 수 표시
- **휴지통 화면 상단**: 프로그레스 바 `████████░░ 8/10`
- **비우기 버튼**: 상태에 따라 텍스트 변경
  - 남은 수 ≥ 휴지통: `[비우기]`
  - 남은 수 < 휴지통: `[N장 무료 삭제 🔓]`
  - 남은 수 = 0: `[광고 보고 비우기]`
- **Plus 구독자 / Grace Period**: 카운터 미표시 (무제한)

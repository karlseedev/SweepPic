## 1. 제품 철학 & 수익화 원칙 ✅

### 구조

```
[사진 정리]        →  [휴지통]       →  [비우기]
항상 무료, 무제한      무료 보관         여기만 게이트
```

- **무료 영역**: 스와이프 삭제, 유사 사진 분석, 자동 정리, 복구 — 전부 무제한
- **게이트**: "비우기"(영구 삭제) 한 곳에만 존재

### 광고 배치 방향

- 정리 행위 **도중**에는 광고 없음 (비교 화면, 뷰어, 그리드)
- 정리 행위 **완료 시점**에 전면 광고 (짝수 회차만, 스킵 가능)
- **대기 화면**에 배너 (분석 중 하단)

| 시점 | 타입 | 빈도 |
|------|------|------|
| 휴지통 비우기 (한도 초과) | 리워드 | 일 3회 |
| 유사사진 삭제 완료 | 전면 | 짝수 회차 |
| 자동정리 휴지통 이동 완료 | 전면 | 짝수 회차 |
| 사진 분석 대기 화면 | 배너 | 매회 |

### 운영 방향

- 사진 정리 행위는 게이트로 막지 않는다 (비우기만 게이트)
- 그리드, 뷰어, 비교 화면 등 탐색 중에는 광고를 넣지 않는다
- 휴지통 복구는 유료화하지 않는다
- Plus 구독자에게는 광고를 보여주지 않는다
- 기존 무료 기능을 나중에 유료로 전환하지 않는다

## 2. 사용자 세그먼트 & 라이프사이클 ✅

### 세그먼트 (가정 — 출시 후 데이터로 검증)

| 유형 | 사진 수 | 일일 정리량 | 게이트 빈도 |
|------|:---:|:---:|:---:|
| 라이트 | ~5,000장 | 1~10장 | 거의 안 걸림 |
| 미디엄 | 5,000~20,000장 | 10~30장 | 가끔 |
| 헤비 | 20,000장+ | 30장+ | 자주~매일 |

### 라이프사이클

```
설치 → Grace Period(3일) → 무료 활성 → 한도 도달 → 전환 or 이탈
                                         ↓
                                    Plus 구독자 → 갱신 or 해지
```

| 단계 | 기간 | 경험 |
|------|------|------|
| 신규 | Day 0~3 | 무제한, 광고 없음 (Grace Period) |
| 무료 활성 | Day 4~ | 일일 10장 무료, 카운터 보임, 전면 광고 노출 |
| 한도 도달 | 수시 | 게이트 시트 → 리워드/Plus 선택지 |
| Plus 전환 | - | 무제한, 광고 없음 |
| 이탈 위험 | 사용 감소 시 | 윈백 대상 |

## 3. 유료 모델 핵심 가치 제안 ✅

> **"제한/광고 없이 바로 정리"**

| 가치 | 무료 | Plus |
|------|------|------|
| 일일 삭제 | 최대 30장 (무료 10 + 리워드 2회) | 무제한 |
| 광고 | 전면(짝수 회차) + 배너(분석 중) + 게이트 시트 | 없음 |
| 대청소 (200장) | 7일 걸림 | 바로 |

- **용량 해방**: 일 30장 캡 해제 → 헤비 유저/대청소 시 결정적
- **귀찮음 제거**: 게이트, 전면 광고, 배너, 카운터 전부 없음

## 4. Free Tier 설계 ✅

### 일일 삭제 한도 (총량 캡 방식)

```
일일 총 삭제 한도 = 무료 N장 + 리워드 M회 × K장
```

| 항목 | 시작값 | 조정 범위 | 비고 |
|------|:---:|:---:|------|
| 무료 | 10장 | 5~15 | 방법 불문 |
| 리워드 1회당 | +10장 | 5~20 | "전체 비우기"가 아닌 추가 장수 |
| 리워드 횟수 | 2회/일 | 1~3 | |
| **일일 최대** | **30장** | **15~45** | 초과 시 Plus만 가능 |

- 매일 자정 리셋
- 모든 숫자는 출시 후 A/B 테스트로 조정
- 자동 정리 실행 횟수: **무제한** (게이트는 비우기에만)
- 한도 초과 시 관용 없음 — 12장이어도 무료 비우기 불가, 게이트 표시
- 리워드 소진 시 → 골든 모먼트 (Plus 전환 유도)

### 게이트 판단

```
비우기 탭
  ↓
휴지통 ≤ 남은 무료 → 게이트 없이 바로 iOS 팝업 → 삭제
휴지통 > 남은 무료 → 게이트 시트 표시
```

### Grace Period

- 설치 후 3일: 광고 없음, 게이트 없음, 카운터 없음 → 무제한
- Day 3 마지막 비우기 후: 종료 예고 알림
- Day 4 첫 게이트: 1회 무료 비우기 쿠폰 (소프트 랜딩)
- Day 5부터: 정상 운영

## 5. 광고 모델 📝

| 광고 | 트리거 | 빈도 |
|-----|--------|------|
| **리워드** | 게이트에서 사용자 선택 | 일 2회 |
| **전면** | 유사사진 삭제 완료 / 자동정리 휴지통 이동 완료 | 짝수 회차 |
| **배너** | 사진 분석 대기 화면 | 매회 |

### 리워드 광고 보상
- 1회 시청 = 일일 삭제 한도 +10장 추가 (전체 비우기가 아님)
- 무료 10 + 리워드 2회 = 일일 최대 30장
- 리워드 횟수는 삭제 성공 후에만 차감 (iOS 팝업 취소 시 소모 안 됨)
- 2회 소진 시 → 골든 모먼트 (Plus 전환 유도)

### 광고 배치 기준
- 정리 행위 **도중**: 광고 없음 (그리드, 뷰어, 비교 화면)
- 정리 행위 **완료 시점**: 전면 광고 (짝수 회차, 스킵 가능)
- **대기 화면**: 배너 (분석 중 하단)
- Grace Period (3일): 모든 광고 없음
- Plus 구독자: 모든 광고 없음

## 6. 구독 상품 & 가격 전략 📝

### Plus 혜택
- 광고 없음, 게이트 없음, 무제한

### 가격

| 플랜 | 정가 | 런칭가 | 비고 |
|------|------|--------|------|
| 월간 | $3.99 | $2.99 | 체험 진입 |
| 연간 | - | $19.99 | 메인 전환 타겟 |

### 무료 체험
- **별도 무료 체험 없음** — Grace Period(3일)가 사실상 무료 체험 역할
- Grace Period 동안 모든 기능 무제한 + 광고 없음 → 앱 가치를 충분히 체험
- 업계는 3일 체험이 표준이나, 별도 결제 없이 같은 효과를 이미 제공

### 미결정 사항
- 연간 정가 (할인 축소 시 목표 가격)
- 단계적 할인 축소 스케줄
- 라이프타임 플랜 포함 여부

## 8. 게이트 & 카운터 UI 📝

### 게이트 시트 (남은 무료 > 0)

```
┌────────────────────────────────────────┐
│  무료 삭제가 N장 남았어요                 │
│  휴지통: M장                            │
│                                        │
│  [N장만 삭제 (무료)]                     │
│  [광고 보고 +10장 삭제]                  │
│  [Plus — 무제한]                        │
│  [나중에]                               │
└────────────────────────────────────────┘
```

### 게이트 시트 (남은 무료 = 0)

```
┌────────────────────────────────────────┐
│  오늘의 무료 삭제를 모두 사용했어요        │
│                                        │
│  [광고 보고 +10장 삭제] (N/2)            │
│  [Plus — 무제한]                        │
│  [내일 다시 올게요]                       │
└────────────────────────────────────────┘
```

### 게이트 시트 (리워드 소진)

```
┌────────────────────────────────────────┐
│  오늘의 무료 삭제를 모두 사용했어요        │
│  (무료 10장 + 리워드 2회 사용 완료)       │
│                                        │
│  [Plus — 무제한]                        │
│  [내일 다시 올게요]                       │
└────────────────────────────────────────┘
```

### 카운터 (듀오링고식)

사용자가 남은 무료 삭제 수를 항상 볼 수 있게 표시:

- **휴지통 탭 아이콘**: 뱃지로 남은 수 표시
- **휴지통 화면 상단**: 프로그레스 바 `████████░░ 8/10`
- **비우기 버튼**: 상태에 따라 텍스트 변경
  - 남은 수 ≥ 휴지통: `[비우기]`
  - 남은 수 < 휴지통: `[N장 무료 삭제 🔓]`
  - 남은 수 = 0, 리워드 남음: `[광고 보고 +10장 삭제]`
  - 남은 수 = 0, 리워드 소진: `[내일 다시 올게요]`
- **Plus 구독자 / Grace Period**: 카운터 미표시 (무제한)

# 저품질 사진 분류 기준 (Low Quality Photo Definition)

**버전**: 1.0
**작성일**: 2026-01-05
**상태**: 최종 확정 (Integrated Standard)

---

## 1. 개요
본 문서는 PickPhoto 앱의 '자동 정리' 기능을 위한 **저품질 사진의 정량적 판별 기준**을 정의한다. 오탐지(False Positive)를 최소화하여 사용자 데이터의 안전성을 보장하면서, 의미 없는 사진을 효율적으로 선별하는 것을 목표로 한다.

## 2. 저품질 사진 판정 기준 (Core Criteria)

다음 3가지 카테고리를 순차적으로 검사하여 하나라도 해당하면 '저품질 후보'로 분류한다.

### 2.1 스크린샷 (Screenshots)
사용자가 정보 전달이나 일시적 보관을 위해 생성한 이미지로, 추억 저장 목적이 낮은 데이터.
- **판별**: PHAsset 메타데이터 확인
- **기준**: `asset.mediaSubtypes.contains(.photoScreenshot)`
- **신뢰도**: 100%

### 2.2 노출 불량 (Bad Exposure)
식별이 불가능할 정도로 너무 어둡거나, 정보 손실이 발생할 정도로 너무 밝은 사진.
- **판별**: 픽셀 휘도(Luminance) 분석 (64x64 다운샘플링 이미지 사용)
- **정량 기준**:
    - **노출 부족 (Dark)**: 평균 휘도 **< 30** (0~255 기준)
    - **과다 노출 (Bright)**: 평균 휘도 **> 230** (0~255 기준)

### 2.3 흔들린 사진 (Blurry)
초점이 맞지 않거나 촬영 시 흔들림으로 인해 피사체 식별이 어려운 사진.
- **판별**: Laplacian 연산의 분산(Variance) 값 계산
- **정량 기준**: **분산 값(Variance) < 100.0**
- **비고**: 텍스트나 경계선이 뭉개진 상태를 수학적으로 판별

---

## 3. 삭제 제외 조건 (Safe Guard / Exclusion)

위 기준에 해당하더라도 다음 조건 중 하나라도 만족하면 **절대로 삭제 대상에 포함하지 않는다.**

| 항목 | 조건 | 이유 |
| :--- | :--- | :--- |
| **즐겨찾기** | `asset.isFavorite == true` | 사용자가 명시적으로 보존 의사를 밝힘 |
| **편집됨** | `asset.hasAdjustments == true` | 사용자의 노력이 들어간 결과물 |
| **심도 효과** | `asset.mediaSubtypes.contains(.depthEffect)` | 의도적인 배경 흐림(Out-focusing) 보호 |
| **인물 보호** | `faceCaptureQuality >= 0.4` | 배경은 흐려도 인물 표정이 선명한 경우 보존 |

---

## 4. 탐색 및 판별 알고리즘 흐름 (Algorithm)

1. **대상 확보**: 라이브러리에서 오래된 사진부터 100장 단위로 Fetch.
2. **1단계 (Fast Filter)**: 메타데이터 기반 검사 (스크린샷 여부, 즐겨찾기, 편집 여부).
3. **2단계 (Pixel Analysis)**: 메타데이터를 통과한 사진 중 심도 효과가 없는 사진 대상.
    - 평균 휘도 계산 → 30 미만 또는 230 초과 시 수집.
    - (위 조건 통과 시) Laplacian 분산 계산 → 100 미만 시 수집.
4. **3단계 (Final Validation)**: 블러로 판정된 사진 중 얼굴이 있다면 품질(0.4) 확인 후 최종 결정.
5. **종료**: 저품질 사진 50장이 수집되면 탐색 중단.

---

## 5. 파라미터 요약 (Summary Table)

| 파라미터 | 정량적 임계값 (Threshold) | 판정 대상 |
| :--- | : :--- | :--- |
| **Darkness** | < 30 (Luminance) | 노출 부족 |
| **Brightness** | > 230 (Luminance) | 과다 노출 |
| **Blur** | < 100.0 (Variance) | 흔들림 |
| **Face Quality** | >= 0.4 (Score) | 인물 사진 보호 |

# 죽은 코드 정리: CleanupConstants.resultMessage()

- 우선순위: 낮음 (프로덕션 영향 없음)
- 발견일: 2026-04-06

## 배경

`CleanupConstants.resultMessage(endReason:foundCount:method:)` 함수가 정의되어 있으나,
미리보기 방식(PreviewGridViewController)으로 전환되면서 UI에서 호출이 빠짐.
현재 테스트 코드에서만 호출되는 죽은 코드 상태.

실제 0장 발견 시 메시지는 `GridViewController+Cleanup.swift`의 `showNoPreviewResultAlert()`가 별도 처리 중.

> - 원본 로그 데이터는 `260103thumbnailLogRaw[N].md`에 분리되어 있습니다
> - 각 파일이 2000줄 초과 시 다음 번호 파일에 저장 (1 → 2 → 3 → ...)
> - 각 로그는 `<!-- LOG_ID: [식별자] -->` 주석으로 검색 가능합니다
> - **새 테스트 추가 시:** 요약은 이 파일에, 원본 로그는 최신 `260103thumbnailLogRaw[N].md`에 따로 저장하세요

## 측정 지표

| 지표 | 설명 |
|------|------|
| velocity | 스크롤 속도 (pt/s) |
| underSized @0.2s | 정지 후 0.2초 시점에 아직 저해상도인 셀 수 |
| underSized @0.6s | 정지 후 0.6초 시점에 아직 저해상도인 셀 수 (0이면 완료) |

### 속도 분류 기준

| 분류 | velocity 범위 | 용도 |
|------|--------------|------|
| **느린** | < 5000 pt/s | 짧은 스크롤 |
| **중간** | 5000~10000 pt/s | 보통 스크롤 |
| **빠른** | > 10000 pt/s | 긴 스크롤 |

# 저품질 사진 분류 기준 (최종 도출)

3개 문서(AutoDel1, AutoDel2, AutoDel3) 통합 분석 결과

---

## 1. 저품질 사진 = 다음 중 하나 이상 해당

| 순위 | 유형 | 판별 방법 | 임계값 | 신뢰도 |
|:---:|------|----------|--------|:------:|
| **1** | 스크린샷 | PHAsset 메타데이터 | `mediaSubtypes.contains(.photoScreenshot)` | 100% |
| **2** | 주머니 속 사진 | 복합 조건 (어둠+단색+흐림) | 밝기<0.10 AND RGB분산<15 AND Laplacian<50 | 95% |
| **3** | 극단적 어두움 | 평균 휘도 | **< 30/255 (약 12%)** | 90% |
| **4** | 극단적 밝음 | 평균 휘도 | **> 240/255 (약 94%)** | 90% |
| **5** | 심한 흐림 | Laplacian Variance | **< 100** | 70% |
| **6** | 단색/무의미 | RGB 표준편차 | **< 10** | 80% |

---

## 2. 예외 처리 (절대 저품질 아님)

다음 조건 중 **하나라도 해당**하면 저품질로 분류하지 않음:

| 조건 | 판별 방법 | 이유 |
|------|----------|------|
| 즐겨찾기 | `asset.isFavorite == true` | 사용자가 중요하다고 표시 |
| 편집됨 | `asset.hasAdjustments` | 사용자가 후보정한 사진 |
| 심도 효과 | `mediaSubtypes.contains(.depthEffect)` | 인물모드는 배경이 의도적으로 흐림 |
| 선명한 얼굴 존재 | `VNFaceCaptureQuality > 0.4` | 배경 흐려도 인물은 선명 |

---

## 3. 최종 임계값 요약

```
┌─────────────────────────────────────────────────────────┐
│                    저품질 판정 기준                      │
├─────────────────────────────────────────────────────────┤
│  [메타데이터]                                           │
│    • 스크린샷 = 저품질                                  │
│                                                         │
│  [밝기 분석] (평균 휘도, 0~255)                         │
│    • < 30   → 극단적 어두움 (저품질)                    │
│    • > 240  → 극단적 밝음 (저품질)                      │
│                                                         │
│  [선명도 분석] (Laplacian Variance)                     │
│    • < 100  → 심한 흐림 (저품질)                        │
│    ※ 단, 심도효과/선명한얼굴 있으면 제외                │
│                                                         │
│  [색상 분석] (RGB 표준편차)                             │
│    • < 10   → 단색/무의미 (저품질)                      │
│                                                         │
│  [복합 조건] (주머니 속 사진)                           │
│    • 밝기<0.10 AND RGB분산<15 AND Laplacian<50          │
└─────────────────────────────────────────────────────────┘
```

---

## 4. 제외한 항목 (이번 버전에서)

| 항목 | 제외 이유 |
|------|----------|
| 저해상도 | 오래된 사진이 모두 걸림, 사용자 불만 예상 |
| 렌즈 가림 | 알고리즘 복잡도 대비 발생 빈도 낮음 |
| 노이즈/대비 | 주관적, 오탐 위험 높음 |
| 눈 감은 사진 | 추가 ML 모델 필요, 복잡도 증가 |

---

## 5. 판별 우선순위 (효율성)

```
1단계: 메타데이터 필터 (빠름, 이미지 로드 불필요)
  ├── 즐겨찾기? → 제외
  ├── 스크린샷? → 저품질 ✓
  └── 심도효과? → 흐림검사 제외

2단계: 썸네일 분석 (64x64 다운샘플)
  ├── 평균 휘도 계산
  │   ├── < 30 → 저품질 ✓
  │   └── > 240 → 저품질 ✓
  │
  └── RGB 표준편차 계산
      └── < 10 → 저품질 ✓

3단계: 선명도 분석 (1단계, 2단계 통과한 것만)
  ├── Laplacian Variance 계산
  │   └── < 100 이면:
  │       ├── 얼굴 품질 검사
  │       └── 선명한 얼굴 없으면 → 저품질 ✓
```

---

## 6. 수치 근거 요약

| 지표 | 임계값 | 근거 |
|------|--------|------|
| 휘도 < 30 | 12% 밝기 | AutoDel3 + 야간촬영 오탐 방지 |
| 휘도 > 240 | 94% 밝기 | AutoDel1 (0.85보다 보수적) |
| Laplacian < 100 | - | PyImageSearch 권장, 3문서 공통 |
| RGB 표준편차 < 10 | - | AutoDel1, 거의 단색 판정 |
| 얼굴 품질 > 0.4 | - | AutoDel3, Vision API 기준 |

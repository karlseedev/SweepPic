# PickPhoto 초기 로딩 성능 최적화 계획 (v6)

## 문제 정의

- **환경**: iPhone 13 Pro, 3,152장 사진
- **증상**: 앱 시작 시 2초 지연, 탭/스크롤 프리즈
- **확정된 원인**: 메인에서 `PHCachingImageManager.requestImage()` 호출이 72ms/회 소요

## 요구사항

| 요구사항 | 설명 |
|----------|------|
| **프리즈 제거** | PHCachingImageManager.requestImage()가 메인에서 **0회** 실행 |
| **첫 화면 0.1초 이내** | degraded라도 콘텐츠가 보이면 OK |
| **디스크 캐시** | 시스템 앱과 동등한 체감을 위해 **필수** |

## 성공 지표 (KPI)

| 지표 | 현재 | 목표 |
|------|------|------|
| 메인에서 PhotoKit 호출 | 17회 | **0회** |
| E0→E1 지연 | 1880ms | **< 150ms** |
| T_firstThumbnailVisible | 2000ms+ | **< 100ms** (degraded 포함) |
| 콜드 런치 캐시 히트 | 0% | **> 90%** (첫 화면, 2nd cold launch 기준)* |
| 프리즈/hitch | 있음 | **없음** |

> *캐시 히트 KPI는 최소 1회 이상 실행하여 캐시가 구축된 상태에서 측정. 첫 설치/캐시 purge 직후는 0%가 정상.

### 1단계: ImagePipeline 구조 개선 (60분)

**목표**: 취소/레이스 안전 + 동시 요청 제한 + PhotoKit 호출을 백그라운드로

**파일**: `ImagePipeline.swift`

#### Cancellable (레이스 안전)

```swift
public protocol Cancellable: AnyObject {
    var isCancelled: Bool { get }
    func cancel()
}

public final class CancellableToken: Cancellable {
    private let lock = NSLock()
    private var _isCancelled = false
    private var _onCancel: (() -> Void)?

    public var isCancelled: Bool {
        lock.withLock { _isCancelled }
    }

    public func cancel() {
        let handler: (() -> Void)?
        lock.lock()
        guard !_isCancelled else { lock.unlock(); return }
        _isCancelled = true
        handler = _onCancel
        _onCancel = nil
        lock.unlock()
        handler?()
    }

    /// 레이스 안전: 이미 cancel됐으면 즉시 실행
    func setOnCancel(_ handler: @escaping () -> Void) {
        lock.lock()
        if _isCancelled {
            lock.unlock()
            handler()
        } else {
            _onCancel = handler
            lock.unlock()
        }
    }
}
```

#### requestImage (백그라운드 전용)

```swift
/// 이미지 요청
/// - PhotoKit 호출은 OperationQueue 내부(백그라운드)에서 실행
/// - completion은 메인 스레드에서 호출
public func requestImage(
    for asset: PHAsset,
    targetSize: CGSize,  // 픽셀 단위 (pt × scale)
    contentMode: PHImageContentMode = .aspectFill,
    completion: @escaping (UIImage?, Bool) -> Void  // (image, isDegraded)
) -> Cancellable {

    let cancellable = CancellableToken()

    requestQueue.addOperation { [weak self] in
        guard let self = self, !cancellable.isCancelled else { return }

        #if DEBUG
        assert(!Thread.isMainThread, "PhotoKit must run on background")
        #endif

        let requestID = self.imageManager.requestImage(
            for: asset,
            targetSize: targetSize,
            contentMode: contentMode,
            options: self.thumbnailOptions
        ) { image, info in
            guard !cancellable.isCancelled else { return }

            let isDegraded = (info?[PHImageResultIsDegradedKey] as? Bool) ?? false

            DispatchQueue.main.async {
                guard !cancellable.isCancelled else { return }
                completion(image, isDegraded)
            }
        }

        // 레이스 안전한 취소 핸들러 설정
        cancellable.setOnCancel { [weak self] in
            self?.imageManager.cancelImageRequest(requestID)
        }
    }

    return cancellable
}

// 동시 요청 제한 (보수적으로 2개 시작)
private let requestQueue: OperationQueue = {
    let queue = OperationQueue()
    queue.name = "com.pickphoto.imagepipeline"
    queue.maxConcurrentOperationCount = 2
    queue.qualityOfService = .userInitiated
    return queue
}()
```

### 2단계: 디스크 캐시 구현 (100분)

**목표**: 콜드 런치에서도 첫 화면 즉시 표시 + 무효화/LRU 정확 동작

**파일**: `Sources/AppCore/Services/ThumbnailCache.swift` (신규)

```swift
import UIKit
import CryptoKit  // iOS 13+ (CommonCrypto 대신)

public final class ThumbnailCache {
    public static let shared = ThumbnailCache()

    private let cacheDirectory: URL
    private let maxCacheSize: Int = 100 * 1024 * 1024  // 100MB
    private let ioQueue = DispatchQueue(label: "com.pickphoto.thumbnailcache", qos: .utility)
    private let scale = UIScreen.main.scale

    private init() {
        let caches = FileManager.default.urls(for: .cachesDirectory, in: .userDomainMask)[0]
        cacheDirectory = caches.appendingPathComponent("Thumbnails", isDirectory: true)
        try? FileManager.default.createDirectory(at: cacheDirectory, withIntermediateDirectories: true)
    }

    // MARK: - Public API

    /// 비동기 로드 + predecode (백그라운드에서 실행, 메인 hitch 방지)
    public func load(
        assetID: String,
        modificationDate: Date?,  // 캐시 무효화용
        size: CGSize,
        completion: @escaping (UIImage?) -> Void
    ) {
        ioQueue.async { [weak self] in
            guard let self = self else {
                DispatchQueue.main.async { completion(nil) }
                return
            }

            let path = self.cachePath(assetID: assetID, modDate: modificationDate, size: size)

            guard FileManager.default.fileExists(atPath: path.path),
                  let image = UIImage(contentsOfFile: path.path) else {
                DispatchQueue.main.async { completion(nil) }
                return
            }

            // LRU: 접근 시 수정일 갱신 (touch)
            try? FileManager.default.setAttributes(
                [.modificationDate: Date()],
                ofItemAtPath: path.path
            )

            // predecode (백그라운드에서)
            let decodedImage: UIImage?
            if #available(iOS 15.0, *) {
                decodedImage = image.preparingForDisplay()
            } else {
                _ = image.cgImage  // 강제 디코딩
                decodedImage = image
            }

            DispatchQueue.main.async {
                completion(decodedImage)
            }
        }
    }

    /// 비동기 저장 (degraded 이미지는 저장 금지)
    public func save(image: UIImage, assetID: String, modificationDate: Date?, size: CGSize) {
        ioQueue.async { [weak self] in
            guard let self = self else { return }
            let path = self.cachePath(assetID: assetID, modDate: modificationDate, size: size)
            if let data = image.jpegData(compressionQuality: 0.8) {
                try? data.write(to: path)
            }
        }
    }

    /// 변경된 asset의 캐시 삭제 (PHPhotoLibraryChangeObserver에서 호출)
    public func invalidate(assetIDs: [String]) {
        ioQueue.async { [weak self] in
            guard let self = self else { return }
            // assetID를 포함하는 파일들 삭제 (해시 역추적 불가하므로 전체 스캔)
            // 또는 별도 매핑 파일 유지
            // 간단 구현: 해당 assetID prefix로 시작하는 파일 삭제
            // (실제로는 해시라 매칭 안됨 → 아래 대안 참고)
        }
    }

    /// 주기적 트림 (앱 시작 시 또는 백그라운드에서 호출)
    public func trimIfNeeded() {
        ioQueue.async { [weak self] in
            guard let self = self else { return }
            self.performLRUTrim()
        }
    }

    // MARK: - Private

    /// Stable key: SHA256(assetID + modDate + width + height + scale)
    /// - modificationDate 포함으로 편집된 사진 자동 무효화
    private func cachePath(assetID: String, modDate: Date?, size: CGSize) -> URL {
        let modString = modDate.map { String($0.timeIntervalSince1970) } ?? "nil"
        let key = "\(assetID)_\(modString)_\(Int(size.width))_\(Int(size.height))_\(Int(scale))"
        let hash = sha256(key)  // 전체 길이 사용
        return cacheDirectory.appendingPathComponent("\(hash).jpg")
    }

    private func sha256(_ string: String) -> String {
        let data = Data(string.utf8)
        let hash = SHA256.hash(data: data)
        return hash.map { String(format: "%02x", $0) }.joined()  // 전체 64자
    }

    private func performLRUTrim() {
        // 수정일 기준 정렬 (접근 시 touch되므로 실제 LRU)
        guard let files = try? FileManager.default.contentsOfDirectory(
            at: cacheDirectory,
            includingPropertiesForKeys: [.contentModificationDateKey, .fileSizeKey]
        ) else { return }

        var totalSize: Int = 0
        var fileInfos: [(url: URL, date: Date, size: Int)] = []

        for file in files {
            guard let values = try? file.resourceValues(forKeys: [.contentModificationDateKey, .fileSizeKey]),
                  let date = values.contentModificationDate,
                  let size = values.fileSize else { continue }
            totalSize += size
            fileInfos.append((file, date, size))
        }

        guard totalSize > maxCacheSize else { return }

        // 오래된 순 정렬 (가장 오래 접근 안 한 것부터)
        fileInfos.sort { $0.date < $1.date }

        for info in fileInfos {
            guard totalSize > maxCacheSize else { break }
            try? FileManager.default.removeItem(at: info.url)
            totalSize -= info.size
        }
    }
}
```

**v5 → v6 변경점**:
- `load()`: 동기 → **비동기** (백그라운드 predecode)
- 캐시 키: modificationDate **포함** (편집 시 자동 무효화)
- LRU: 접근 시 **touch** (수정일 갱신)
- 해시: CommonCrypto → **CryptoKit**, 전체 64자 사용

**구캐시 회수 정책**: 사진 편집 시 modificationDate가 변경되어 새 캐시 파일이 생성되고, 구 캐시 파일은 LRU trim에 의해 자연 회수됨. PHPhotoLibraryChangeObserver 기반 즉시 삭제는 향후 최적화로 분리.

### 3단계: PhotoCell 수정 (50분)

**목표**: 비동기 디스크 캐시 연동 + 백그라운드 요청 + 취소 안정화

**파일**: `PhotoCell.swift`

```swift
private var currentCancellable: Cancellable?
private var currentCacheLoadID: UUID?  // 캐시 로드 취소용

func configure(asset: PHAsset, isTrashed: Bool, targetSize: CGSize) {
    let assetID = asset.localIdentifier
    let modDate = asset.modificationDate
    self.currentAssetID = assetID

    // 이전 요청 취소
    currentCancellable?.cancel()
    currentCancellable = nil
    currentCacheLoadID = nil

    // targetSize를 픽셀 단위로 변환
    let scale = UIScreen.main.scale
    let pixelSize = CGSize(width: targetSize.width * scale, height: targetSize.height * scale)

    // 캐시 로드 ID (비동기 캐시 결과 검증용)
    let loadID = UUID()
    currentCacheLoadID = loadID

    // 1) 디스크 캐시에서 비동기 로드 (백그라운드 predecode)
    ThumbnailCache.shared.load(
        assetID: assetID,
        modificationDate: modDate,
        size: pixelSize
    ) { [weak self] cachedImage in
        // 메인 스레드
        guard self?.currentAssetID == assetID,
              self?.currentCacheLoadID == loadID else { return }

        if let cachedImage = cachedImage {
            self?.imageView.image = cachedImage
            return  // 캐시 히트 → PhotoKit 요청 불필요
        }

        // 2) 캐시 미스 → ImagePipeline 요청
        self?.requestFromPipeline(asset: asset, pixelSize: pixelSize, modDate: modDate)
    }
}

private func requestFromPipeline(asset: PHAsset, pixelSize: CGSize, modDate: Date?) {
    let assetID = asset.localIdentifier

    currentCancellable = ImagePipeline.shared.requestImage(
        for: asset,
        targetSize: pixelSize
    ) { [weak self] image, isDegraded in
        guard self?.currentAssetID == assetID else { return }

        if let image = image {
            self?.imageView.image = image

            // degraded가 아닌 최종 이미지만 캐시에 저장
            if !isDegraded {
                ThumbnailCache.shared.save(
                    image: image,
                    assetID: assetID,
                    modificationDate: modDate,
                    size: pixelSize
                )
            }
        }
    }
}

override func prepareForReuse() {
    super.prepareForReuse()
    currentCancellable?.cancel()
    currentCancellable = nil
    currentCacheLoadID = nil
    currentAssetID = nil
    imageView.image = nil
}
```

**v5 → v6 변경점**:
- `ThumbnailCache.load()`: 동기 → **비동기** (completion 기반)
- `currentCacheLoadID`: 비동기 캐시 로드 **취소/검증용**
- `modificationDate` 전달: 캐시 **무효화** 지원

```swift
private var hasPreheatedInitialScreen = false

/// 첫 화면 프리히트 (viewDidAppear 이후 호출 - visible 보장)
private func preheatInitialScreen() {
    guard !hasPreheatedInitialScreen else { return }
    hasPreheatedInitialScreen = true

    // 정확한 pixelSize
    let scale = UIScreen.main.scale
    let targetSize = CGSize(
        width: thumbnailSize().width * scale,
        height: thumbnailSize().height * scale
    )

    // viewDidAppear 이후이므로 visible indexPaths 확실히 존재
    let visibleIndexPaths = collectionView.indexPathsForVisibleItems
    guard !visibleIndexPaths.isEmpty else {
        // 만약을 위한 fallback
        hasPreheatedInitialScreen = false
        return
    }

    // +1 화면 반경
    let allIndexPaths = extendIndexPaths(visibleIndexPaths, by: 21)

    let assets = allIndexPaths.compactMap { dataSourceDriver.asset(at: $0) }
    guard !assets.isEmpty else { return }

    // 백그라운드에서 프리히트
    DispatchQueue.global(qos: .userInitiated).async {
        ImagePipeline.shared.preheatAssets(assets, targetSize: targetSize)
    }
}

override func viewDidAppear(_ animated: Bool) {
    super.viewDidAppear(animated)

    // visible indexPaths가 확실히 채워진 시점
    preheatInitialScreen()
}

/// loadData는 프리히트 호출 제거
private func loadData() {
    dataSourceDriver.reloadData { [weak self] in
        guard let self = self else { return }

        self.updateEmptyState()
        self.collectionView.reloadData()
        self.collectionView.layoutIfNeeded()

        // 프리히트는 viewDidAppear에서 호출
        // (여기서 호출하면 visible이 비어있을 수 있음)

        // scrollToItem 등...
    }
}
```

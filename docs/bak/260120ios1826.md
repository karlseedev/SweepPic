# iOS 18/26 버전 분기 처리 보완 계획

## 배경

리팩토링 진행 중 iOS 18과 iOS 26 버전 분기 처리가 누락된 부분이 발견되었습니다.
이 문서는 누락된 분기 처리를 보완하기 위한 상세 계획입니다.

---

|-------|------|------|

## 원칙: 조건부 생성 (생성 후 숨기기 X)

**잘못된 방식:**
```swift
// ❌ 만들어놓고 iOS 26에서 숨기기
lazy var floatingOverlay: FloatingOverlay = { ... }()
if #available(iOS 26.0, *) {
    floatingOverlay.isHidden = true
}
```

**올바른 방식:**
```swift
// ✅ 처음부터 분기해서 다른 UI 생성
if #available(iOS 26.0, *) {
    setupSystemNavigationBar()
} else {
    setupFloatingOverlay()
}
```

## 발견된 문제점

### 문제 1: BaseGridViewController.viewWillAppear iOS 분기 누락

**현재 코드 (143~146줄):**
```swift
override func viewWillAppear(_ animated: Bool) {
    super.viewWillAppear(animated)
    configureFloatingOverlay()  // iOS 버전 무관 호출
}
```

**문제:**
- iOS 26+에서 네비바 hidden 상태 복원이 없음
- push/pop 전환 후 네비바가 잘못된 상태일 수 있음

**영향 범위:**
- GridViewController (상속)
- AlbumGridViewController (상속)
- 향후 TrashAlbumViewController (Phase 5 완료 후)

**참고 패턴 (TrashAlbumViewController 181~196줄):**
```swift
override func viewWillAppear(_ animated: Bool) {
    super.viewWillAppear(animated)

    if useFloatingUI {
        // iOS 16~25: 시스템 바 숨김 유지
        navigationController?.setNavigationBarHidden(true, animated: animated)
        configureFloatingOverlayForTrash()
    } else {
        // iOS 26+: 시스템 바 표시
        navigationController?.setNavigationBarHidden(false, animated: animated)
        setupSystemNavigationBar()
    }
    // ...
}
```

---

### 문제 2: GridViewController.setupSystemNavigationBar 오버라이드 누락

**현재 상태:**
- BaseGridViewController.setupSystemNavigationBar()는 타이틀만 설정 (201~204줄)
- GridViewController에서 오버라이드하지 않음
- iOS 26에서 **Select 버튼이 없음**

**참고:**
- `selectButtonTapped()` 메서드는 GridSelectMode.swift에 이미 존재 (160줄)
- `exitSelectModeSystemUI()`에서 Select 버튼을 설정하는 코드 존재 (143줄)
- 하지만 초기 상태에서는 Select 버튼이 설정되지 않음

### 문제 3: AlbumGridViewController.viewWillAppear iOS 분기 불완전

**현재 코드 (98~112줄):**
```swift
override func viewWillAppear(_ animated: Bool) {
    super.viewWillAppear(animated)

    if useFloatingUI {
        configureFloatingOverlayForAlbum()
    }
    // else 분기 없음! iOS 26+ 처리 누락
    // ...
```

**문제:**
- iOS 26+에서 네비바 hidden 상태 설정이 없음
- super.viewWillAppear()가 BaseGridViewController를 호출하지만, Base에도 분기가 없음

**참고:** 이 문제는 문제 1이 해결되면 자동으로 해결될 수 있음 (super 호출로 Base의 분기 상속)

### 문제 4: AlbumsViewController.viewWillAppear iOS 분기 누락

**현재 코드 (103~111줄):**
```swift
override func viewWillAppear(_ animated: Bool) {
    super.viewWillAppear(animated)
    loadData()
    configureFloatingOverlayForAlbums()  // iOS 버전 무관 호출
```

**문제:**

- configureFloatingOverlayForAlbums()는 iOS 26에서 floatingOverlay가 nil이므로 조기 리턴됨
- navigationItem.title = "앨범"은 setupUI()에서 설정되어 있음 (132줄) ✅

**참고:** AlbumsViewController는 BaseGridViewController를 상속받지 않음 (리팩토링 대상 아님)

## 수정 계획

### Task 1: BaseGridViewController.viewWillAppear 수정

**파일:** `PickPhoto/PickPhoto/Features/Grid/BaseGridViewController.swift`

**수정 위치:** 143~146줄

**현재:**
```swift
override func viewWillAppear(_ animated: Bool) {
    super.viewWillAppear(animated)
    configureFloatingOverlay()
}
```

**변경:**
```swift
override func viewWillAppear(_ animated: Bool) {
    super.viewWillAppear(animated)

    // iOS 버전별 네비바 상태 설정
    if useFloatingUI {
        // iOS 16~25: 시스템 바 숨김 (FloatingOverlay 사용)
        navigationController?.setNavigationBarHidden(true, animated: animated)
        configureFloatingOverlay()
    } else {
        // iOS 26+: 시스템 바 표시
        navigationController?.setNavigationBarHidden(false, animated: animated)
    }
}
```

**영향:**
- GridViewController: super 호출로 자동 적용
- AlbumGridViewController: super 호출로 자동 적용

---

### Task 2: GridViewController.setupSystemNavigationBar 오버라이드 추가

**파일:** `PickPhoto/PickPhoto/Features/Grid/GridViewController.swift`

**추가 위치:** setupFloatingOverlay() 근처 또는 MARK: - iOS 버전별 UI 섹션

**추가할 코드:**
```swift
// MARK: - iOS 26+ 시스템 UI

/// iOS 26+: 시스템 네비게이션 바 설정
/// - 타이틀: "사진보관함"
/// - 우측: Select 버튼
@available(iOS 26.0, *)
override func setupSystemNavigationBar() {
    super.setupSystemNavigationBar()

    // Select 버튼 추가
    navigationItem.rightBarButtonItem = UIBarButtonItem(
        title: "Select",
        style: .plain,
        target: self,
        action: #selector(selectButtonTapped)
    )
}
```

**참고:**
- `selectButtonTapped()` 메서드는 GridSelectMode.swift에 이미 존재 (160줄)
- Select 모드 종료 시 `exitSelectModeSystemUI()`에서 동일한 버튼을 설정함 (139~144줄)

# 보관함/앨범 뷰어 휴지통 사진 표시 일관성 수정

## 문제
- **그리드**: 휴지통 사진이 표시됨 (마룬 딤드 오버레이)
- **뷰어**: 휴지통 사진이 제외됨 (스와이프로 건너뛰어짐)
- **논리적 불일치**: 그리드에서 보이면 뷰어에서도 보여야 함

## 해결 방안
- 보관함/앨범 뷰어에서도 휴지통 사진 표시
- 휴지통 사진은 **마룬(#800000) 10px 두께 테두리**로 표시
- 휴지통 사진에서는 **복구 버튼만** 표시 (삭제 버튼 대신)
- 휴지통 탭 뷰어에서는 테두리 표시 안 함

**파일**: `PickPhoto/Features/Viewer/ViewerCoordinator.swift`

`rebuildIndexMapping()`에서 `.normal` 모드일 때 조기 리턴 (O(n) 스캔 스킵):
```swift
// 라인 237 부근, trashedIDs.isEmpty 체크 앞에 추가
if viewerMode == .normal {
    // .normal 모드에서는 휴지통 사진도 표시 (그리드와 일관성)
    indexMapping = .identity(totalOriginal: totalOriginal)
    return
}
```

### Phase 4: VideoPageViewController 수정

### Phase 5: ViewerViewController 수정
**파일**: `PickPhoto/Features/Viewer/ViewerViewController.swift`

1. `createPageViewController(at:)`에서 테두리 표시 조건:
   ```swift
   // 보관함(.normal)에서만 테두리 표시, 휴지통 탭에서는 표시 안 함
   let showBorder = (mode == .normal) && coordinator.isTrashed(at: index)
   return PhotoPageViewController(asset: asset, index: index, showTrashedBorder: showBorder)
   ```

2. **삭제/복구 버튼 모두 생성** (viewDidLoad에서):
   - `.normal` 모드: 삭제 버튼 + 복구 버튼 둘 다 생성
   - `isHidden`으로 토글 (같은 위치에 배치)
   ```swift
   // 두 버튼 모두 생성, 초기 상태에서 하나만 보이게
   private lazy var deleteButton: UIButton = { ... }()
   private lazy var restoreButton: UIButton = { ... }()

   private func updateToolbarForCurrentPhoto() {
       let isTrashed = coordinator.isTrashed(at: currentIndex)
       if mode == .normal {
           deleteButton.isHidden = isTrashed
           restoreButton.isHidden = !isTrashed
       }
   }
   ```

3. **복구 시 모드별 분기 처리**:
   - `.trash` 모드: 기존 동작 (목록에서 사라지므로 다음 사진으로 이동)
   - `.normal` 모드: 제자리 유지, 테두리 제거 + 버튼 교체만
   ```swift
   @objc private func restoreButtonTapped() {
       // ... 복구 요청 ...

       if mode == .trash {
           // 기존: 다음 사진으로 이동
           moveToNextAfterDelete()
       } else {
           // .normal: 제자리에서 UI만 업데이트
           updateCurrentPageTrashedState(isTrashed: false)
           updateToolbarForCurrentPhoto()
       }
   }
   ```

4. **현재 페이지 테두리 즉시 업데이트**:
   ```swift
   private func updateCurrentPageTrashedState(isTrashed: Bool) {
       guard let currentVC = pageViewController.viewControllers?.first else { return }
       if let photoVC = currentVC as? PhotoPageViewController {
           photoVC.updateTrashedState(isTrashed: isTrashed)
       } else if let videoVC = currentVC as? VideoPageViewController {
           videoVC.updateTrashedState(isTrashed: isTrashed)
       }
   }
   ```

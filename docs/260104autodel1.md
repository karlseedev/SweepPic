# 저품질 사진 자동 선별 기능

## 1. 개요

**목표**: 그리드에서 "정리" 버튼을 누르면 저품질 사진을 자동 분석하여 선택하는 기능

**UI 동작 흐름**:
1. 그리드 네비바/플로팅 UI에 "정리" 버튼 추가 (Select 옆)
2. 버튼 탭 → 분석 시작 (프로그레스 표시)
3. 분석 완료 → Select 모드 진입 + 저품질 사진 자동 선택
4. 사용자가 검토 후 삭제 또는 취소

---

## 2. 저품질 사진 정의

### 2.1 삭제 대상 유형

| 유형 | 설명 | 발생 원인 |
|------|------|----------|
| **흐릿한 사진** | 초점이 안 맞거나 흔들린 사진 | 카메라 흔들림, 피사체 움직임, 초점 오류 |
| **과다 노출** | 너무 밝아서 디테일이 날아간 사진 | 밝은 환경에서 노출 보정 실패 |
| **노출 부족** | 너무 어두워서 내용 파악 불가 | 어두운 환경, 주머니 속 촬영 |
| **렌즈 가림** | 손가락이나 물체가 렌즈를 가린 사진 | 잘못된 그립, 케이스 간섭 |
| **단색/무의미** | 검은 화면, 천장, 바닥 등 | 실수로 버튼 누름, 아기가 만짐 |

### 2.2 삭제 대상이 아닌 사진 (제외 필요)

- **스크린샷**: 메타데이터로 필터링
- **의도적 야경/실루엣**: 어둡지만 예술적 의도
- **문서 촬영**: 단색 배경이지만 유용
- **얼굴 포함 사진**: 더 보수적으로 판정

---

## 3. 감지 기준 및 임계값

### 3.1 흐릿함 감지 (Laplacian Variance)

```
원리: Laplacian 연산자로 이미지의 2차 미분 계산
     - 고분산 = 선명한 엣지 = 선명한 이미지
     - 저분산 = 적은 엣지 = 흐릿한 이미지
```

| Laplacian 분산 | 판정 |
|----------------|------|
| **< 50** | 매우 흐릿함 (주머니 속 가능성) |
| **< 100** | 흐릿함 (삭제 후보) |
| **100 ~ 500** | 약간 흐릿함 |
| **> 500** | 선명함 |

**주의**: 이미지 480px 리사이즈 후 계산, 데이터셋마다 튜닝 필요

### 3.2 노출 문제 감지 (평균 밝기)

| 평균 밝기 (0~1) | 판정 |
|-----------------|------|
| **< 0.10** | 매우 어두움 (주머니 속) |
| **< 0.15** | 노출 부족 |
| **0.15 ~ 0.85** | 정상 |
| **> 0.85** | 과다 노출 |
| **> 0.90** | 매우 밝음 (하얀 화면) |

**히스토그램 분석 (보조):**
- 노출 부족: 어두운 영역(0-50)에 픽셀 40% 이상
- 과다 노출: 밝은 영역(200-255)에 픽셀 65% 이상

### 3.3 렌즈 가림 감지

| 패턴 | 특징 | 원인 |
|------|------|------|
| 한쪽 모서리 어두움 | 한 방향만 급격히 어두움 | 손가락 가림 |
| 전체 가장자리 어두움 | 비네팅 패턴 | 케이스 간섭 |

**알고리즘:**
```
1. 이미지를 9개 영역 분할 (3x3 그리드)
2. 각 영역 평균 밝기 계산
3. 모서리 밝기 < 중앙 × 0.4 → 가림 판정
4. 한 모서리만 < 0.15 → 손가락 가림
```

### 3.4 단색/무의미 감지 (RGB 분산)

| RGB 표준편차 | 판정 |
|--------------|------|
| **< 10** | 거의 단색 (삭제 후보) |
| **10 ~ 20** | 저대비 |
| **> 20** | 정상 |

### 3.5 주머니 속 사진 (복합 조건)

```
AND 조건 모두 충족 시 주머니 속 판정:
- 평균 밝기 < 0.10
- RGB 표준편차 < 15
- Laplacian 분산 < 50
- 모서리/중앙 밝기 차이 < 0.05
```

### 3.6 임계값 요약

| 지표 | 임계값 | 조건 |
|------|--------|------|
| Laplacian 분산 | < 100 | 흐릿함 |
| 평균 밝기 | < 0.15 | 노출 부족 |
| 평균 밝기 | > 0.85 | 과다 노출 |
| RGB 표준편차 | < 10 | 단색 |
| 모서리/중앙 비율 | < 0.4 | 렌즈 가림 |

## 4. 안전장치

1. **삭제 전 사용자 확인**: 자동 삭제 금지, 선택 후 검토
2. **스크린샷 제외**: 메타데이터로 필터링
3. **얼굴 포함 사진 주의**: 얼굴 감지 시 보수적 판정
4. **최근 사진 주의**: 24시간 이내는 더 보수적으로
5. **임계값 조정**: 사용자 민감도 설정 (향후)

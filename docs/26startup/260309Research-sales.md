## 1. 예상 매출 종합표

### 전제 조건
| 항목 | 값 |
|------|---|
| 총 광고선전비 | 2,250만원 (정부지원, 10개월) |
| 외주용역비 (영상) | 300만원 |
| 유료 설치 | 6,842건 |
| ASO 오가닉 배수 | ×1.4 (+40%) |
| **총 설치 추정** | **~9,600건** |
| 구독 가격 (월) | ₩4,400 ($2.99, App Store Tier) |
| 구독 가격 (연) | ₩29,000 ($19.99, App Store Tier) |
| 월구독:연구독 비율 | 60:40 |
| Apple 수수료 | 30% (1년차), 15% (2년차~) |
| 수익 모델 | Freemium (구독 + 리워드 광고) |

### 시나리오별 매출 추정

| | 보수적 (전환 2%) | 중간 (전환 3.5%) | 낙관적 (전환 5%) |
|--|-----------------|-----------------|-----------------|
| 구독 전환 | 192명 | 336명 | 480명 |
| 구독 매출/월 | ~49만원 | ~85만원 | ~121만원 |
| 광고 매출/월 | ~8만원 | ~17만원 | ~25만원 |
| **월 매출** | **~57만원** | **~102만원** | **~146만원** |
| **연 매출** | **~680만원** | **~1,220만원** | **~1,750만원** |

### 구독 매출 산출 상세 (중간 시나리오 기준)
- 총 설치 9,600건 × 전환율 3.5% = 336명 구독
- 월구독 (60%): 202명 × ₩4,400 × 0.70 = ₩621,544/월
- 연구독 (40%): 134명 × ₩29,000/12 × 0.70 = ₩226,683/월
- **구독 매출 합계: ~848,000원/월**

## 2. 광고 채널별 예상 설치 수

### 광고비 배분 (글로벌 집중 + 인스타 릴스)

| 채널 | 금액 | CPI | 설치 수 | 비고 |
|------|------|-----|--------|------|
| Apple Search Ads (국내) | 400만원 | ₩1,500 | 2,667건 | CPI 최저, 고의도 사용자 |
| Apple Search Ads (글로벌: 미국·일본·유럽) | 1,150만원 | ₩3,500 | 3,286건 | 글로벌 핵심 채널 |
| 인스타그램 릴스 (한국/글로벌) | 400만원 | ₩4,500 | 889건 | 브랜드 인지도 + 대량 도달 |
| ASO 최적화 (키워드 분석, 다국어 현지화) | 300만원 | - | 오가닉 +2,737건 | 유료설치 대비 ×1.4 |
| **합계** | **2,250만원** | | **유료 6,842건 + 오가닉 2,737건 = 9,579건** | |

### 채널 선택 근거
- **Apple Search Ads 비중 69%**: iOS 앱에 가장 효율적. 앱을 이미 찾고 있는 고의도 사용자에게 도달. 국내 CPI ₩1,500, 글로벌 CPI ₩3,500
- **인스타 릴스 비중 18%**: 앱을 모르는 잠재 사용자에게 대규모 도달. 외주 제작 영상 소재 활용. 비팔로워 55% 노출로 신규 사용자 발굴에 강함
- **ASO 비중 13%**: 비용 대비 효과 최고. 오가닉 유입은 유료 대비 LTV 20~40% 높음. 복리 효과 발생
- **구글 앱캠페인 미편성 사유**: 현재 iOS 전용 앱. Android 출시 후 별도 편성 예정
- **랜딩페이지 미편성 사유**: 앱스토어 자체가 랜딩 역할. 초기에는 직접 전환 채널에 집중

## 10. 릴스 광고 소재 제작 가이드

### 기술 스펙
- 비율: 9:16 (세로형 필수)
- 길이: 6~30초 권장
- 해상도: 1080×1920 이상

### 핵심 원칙
| 원칙 | 상세 |
|------|------|
| 3초 법칙 | 첫 3초에 주목을 끌어야 함. 3초 유지율 60% 이상이면 도달 5~10배 차이 |
| 진정성 우선 | 스튜디오급보다 UGC/실사용 느낌이 성과 높음 |
| 텍스트 오버레이 | 캡션/자막 필수 (음소거 시청 비율 높음) |
| 후크 → 데모 → CTA | 3초 후크(문제 제시) → 앱 사용 데모 → "지금 다운로드" |

### SweepPic 소재 아이디어
1. **Before/After**: "사진 5만 장 → 스와이프로 3분 만에 정리"
2. **문제 제시형**: "사진첩이 이런 상태라면..." → 앱 시연
3. **ASMR/만족감**: 스와이프 삭제 제스처의 시각적 만족감
4. **비교형**: "기본 사진 앱 vs SweepPic" 속도 비교

## 12. VAT 자부담 정리

### 핵심
- 정부지원금으로 VAT 충당 **불가** (세부관리기준 명시)
- 인건비는 **면세** (급여는 부가세 과세 대상 아님)
- 부가세 신고 시 **전액 환급 가능** (시차 1~6개월)

### 비목별 VAT 부담 (정부지원금 기준)
| 비목 | 금액 | VAT 적용 | VAT 부담액 |
|------|------|---------|-----------|
| 인건비 | 1,750만원 | 면세 | 0원 |
| 광고선전비 | 2,250만원 | 과세 | 225만원 |
| 외주용역비 | 300만원 | 과세 | 30만원 |
| 창업활동비 | 100만원 | 과세 | ~10만원 |
| **합계** | **4,400만원** | | **약 265만원** |

### 사업계획서 표기법
- 공급가액 기준으로 작성
- 산출근거에 "(VAT 별도, 자부담 예정)" 명시 → 실무 이해도 입증

## `1. 문제 인식 (Problem)`

### `1. 창업 아이템의 필요성`
**사진 보관량 증가와 정리 수요**

- 전 세계적으로 연간 약 2조 장의 사진이 촬영되고 있으며¹⁾, Google Photos에는 15억 명의 사용자가 9조 장 이상의 사진과 동영상을 저장하고 있음²⁾.
- 사진이 쌓이면 정리가 필요하지만, 불필요한 사진을 골라내려면 한 장씩 직접 확인해야 함. 같은 장면의 연속 촬영에서는 표정·눈감김 등 미세한 차이까지 비교해야 하므로 피로도가 더욱 높음. 실제로 스마트폰 사용자의 55%가 사진 수에 압도감을 느끼고 있으며³⁾, 60%는 정리를 고려하면서도 실행에 옮기지 못하고 있음⁴⁾.
- 한편 사용자는 자동 삭제를 신뢰하지 않으며, 사진을 직접 보면서 판단하고 싶어 함⁵⁾. 직접 보면서 정리하는 과정 자체에서 만족감을 느낀다는 연구 결과도 있음⁶⁾.
- 이러한 수요는 결국 사진 정리 앱 시장을 글로벌 기준 연간 소비자 지출 수천억 원 규모로 성장시켰음⁷⁾.

**기존 솔루션의 현황과 한계**

사진 정리 시장의 기존 솔루션은 크게 수동정리형, 자동정리형, 갤러리형의 세 유형으로 나뉘는데, 각각 아래와 같은 한계가 있음.

- 수동정리형(Slidebox, SwipeWipe 등): 사진을 한 장씩 넘기며 판단하는 방식으로, 대량의 사진을 빠르게 훑어보며 정리하기 어려움. 일부 앱은 유사 사진을 추출하여 선별하는 기능을 제공하나, 축소된 썸네일로는 표정 등 얼굴의 미세한 차이를 비교하기 어려움.

- 자동정리형(Cleanup, Cleaner Guru 등): AI가 블러·중복 등 기술적 결함은 감지하지만, 표정·구도 등 사용자의 주관적 선호는 판단하지 못함. 결국 최종 선별은 사용자 몫임.

- 갤러리형(HashPhotos, MyPics 등): 시간순·앨범별 탐색에 최적화되어 있으나, 삭제마다 개별 선택과 확인을 거쳐야 하여 정리 전용 기능이 미흡함.

**미충족 수요로 인한 개발 필요성**

- 높은 사용자 수요에 따라 다양한 사진 정리 앱이 등장하였으나, 사진 앱 카테고리의 30일 리텐션은 1.5%로 전 카테고리 최하위이며⁸⁾, 월 매출 1위 앱조차 56만 건 리뷰 분석에서 신뢰도 31.2점(100점 만점)에 그치는 등⁹⁾ 사용자 만족도는 여전히 낮음. 수천억 원 규모의 시장과 낮은 사용자 만족도 사이의 격차는 시장 진입의 기회임이 분명함.

`정부지원사업비 집행 계획 기재`

**SweepPic — 탐색과 정리를 통합한 사진 관리 솔루션**

| 항목 | 내용 |
|------|------|
| 제품명 | SweepPic |
| 개요 | 사진 탐색과 정리를 하나의 흐름으로 통합한 사진 관리 앱 |
| 제품 형태 | 모바일 애플리케이션 (iPhone / iPad / Android) |
| 핵심 가치 | ① 얼굴 크롭·비교를 통한 유사사진 수동 정리 편의성 극대화 ② 품질점수 기반 3단계 옵션으로 보다 정밀한 자동 정리 기능 제공 ③ 갤러리 인터페이스와 썸네일 즉시 삭제로 차별화된 정리 경험 제공 |


**기존 한계별 해결 전략**

그리드 스와이프 즉시 삭제, 유사사진 얼굴 크롭·비교, 품질점수 기반 3단계 자동정리 등의 기능을 통해 탐색과 정리가 분리된 기존 앱의 문제점을 해결.

| 유형 | 기존 앱의 한계 | SweepPic 솔루션 |
|------|----------|-------------------|
| 갤러리형 | 빠른 대량정리 불편, 삭제마다 확인 필요 | 그리드에서 가로 스와이프로 확인 없이 즉시 삭제, 삭제대기함으로 복구 가능 |
| 수동정리형 | 축소 썸네일로는 얼굴 미세 차이 비교 어려움 | 동일 인물 얼굴을 자동 크롭·확대하여 표정 비교, 선택 삭제 |
| 자동정리형 | AI가 일괄적 기준으로만 저품질 판정 | 품질점수기반 3단계(완화/기본/강화)로 분류, 사용자가 정리 범위를 직접 조절 |

**핵심 기술 및 구현 현황**

| 기술 과제 | 해결 방식 |
|----------|----------|
| 5만 장 이상의 라이브러리에서 끊김 없는 그리드 탐색 | 우선순위 큐 기반 비동기 썸네일 로딩, 인접 행 프리히트, 스크롤 중 저화질 자동 전환, LRU 캐싱을 조합하여 120fps 스크롤 유지 |
| 연속 촬영된 유사 사진의 자동 그룹화 | 이미지 특징 벡터 간 유사도 비교로 연속 촬영 사진을 자동 그룹화(최소 3장). FIFO 큐로 분석 요청을 관리하며, 기기 과열 시 동시 처리 수를 자동 조절 |
| 유사 사진 그룹 내 동일 인물 식별 및 비교 | 얼굴 감지 → 품질 평가 → 임베딩 추출 → 코사인 유사도 비교로 동일 인물을 자동 매칭. 인물별 얼굴을 크롭·확대하여 2열 그리드로 나열, 표정 차이를 직관적으로 비교 가능 |
| 저품질 사진의 자동 감지 및 중요 사진 보호 | 4단계 순차 필터 설계: ① 메타데이터 필터(즐겨찾기·편집됨 등 보호) → ② 노출 분석(극단 어두움/밝음) → ③ 블러 분석(Laplacian Variance 기반) → ④ Safe Guard(Portrait 모드·선명한 얼굴 보호로 오탐 방지) |
| 사용자마다 다른 정리 기준에 대응 | 품질 점수를 완화(30점 이하)/기본(31~40점)/강화(41~50점)로 3단계 분류. "더 보기/덜 보기" 버튼으로 사용자가 정리 범위를 직접 조절 |

## `1. 문제 인식 (Problem)`

### `1. 창업 아이템의 필요성`
**사진 보관량 증가와 정리 수요**

- 전 세계적으로 연간 약 2조 장의 사진이 촬영되고 있으며¹⁾, Google Photos에는 15억 명의 사용자가 9조 장 이상의 사진과 동영상을 저장하고 있음²⁾.
- 사진이 쌓이면 정리가 필요하지만, 불필요한 사진을 골라내려면 한 장씩 직접 확인해야 함. 같은 장면의 연속 촬영에서는 표정·눈감김 등 미세한 차이까지 비교해야 하므로 피로도가 더욱 높음. 실제로 스마트폰 사용자의 55%가 사진 수에 압도감을 느끼고 있으며³⁾, 60%는 정리를 고려하면서도 실행에 옮기지 못하고 있음⁴⁾.
- 한편 사용자는 자동 삭제를 신뢰하지 않으며, 사진을 직접 보면서 판단하고 싶어 함⁵⁾. 직접 보면서 정리하는 과정 자체에서 만족감을 느낀다는 연구 결과도 있음⁶⁾.
- 이러한 수요는 결국 사진 정리 앱 시장을 글로벌 기준 연간 소비자 지출 수천억 원 규모로 성장시켰음⁷⁾.

**기존 솔루션의 현황과 한계**

사진 정리 시장의 기존 솔루션은 크게 수동정리형, 자동정리형, 갤러리형의 세 유형으로 나뉘는데, 각각 아래와 같은 한계가 있음.

- 수동정리형(Slidebox, SwipeWipe 등): 사진을 한 장씩 넘기며 판단하는 방식으로, 대량의 사진을 빠르게 훑어보며 정리하기 어려움. 일부 앱은 유사 사진을 추출하여 선별하는 기능을 제공하나, 축소된 썸네일로는 표정 등 얼굴의 미세한 차이를 비교하기 어려움.

- 자동정리형(Cleanup, Cleaner Guru 등): AI가 블러·중복 등 기술적 결함은 감지하지만, 표정·구도 등 사용자의 주관적 선호는 판단하지 못함. 결국 최종 선별은 사용자 몫임.

- 갤러리형(HashPhotos, MyPics 등): 시간순·앨범별 탐색에 최적화되어 있으나, 삭제마다 개별 선택과 확인을 거쳐야 하여 정리 전용 기능이 미흡함.

**미충족 수요로 인한 개발 필요성**

- 높은 사용자 수요에 따라 다양한 사진 정리 앱이 등장하였으나, 사진 앱 카테고리의 30일 리텐션은 1.5%로 전 카테고리 최하위이며⁸⁾, 월 매출 1위 앱조차 56만 건 리뷰 분석에서 신뢰도 31.2점(100점 만점)에 그치는 등⁹⁾ 사용자 만족도는 여전히 낮음. 수천억 원 규모의 시장과 낮은 사용자 만족도 사이의 격차는 시장 진입의 기회임이 분명함.

**기존 한계별 차별화 전략**

SweepPic는 그리드 스와이프 즉시 삭제, 유사사진 얼굴 크롭·비교, 품질점수 기반 3단계 자동정리 등의 기능을 독자적으로 개발하여, 탐색과 정리가 분리된 기존 앱의 한계를 해결하고 차별화된 정리 경험을 구현함.

| 유형 | 기존 앱의 한계 | SweepPic 차별화 솔루션 |
|------|----------|-------------------|
| 수동정리형 | 축소 썸네일로는 얼굴 미세 차이 비교 어려움 | 동일 인물 얼굴을 자동 크롭·확대하여 전용 비교 화면에서 표정 비교, 선택 삭제 |
| 자동정리형 | AI가 일괄적 기준으로만 저품질 판정 | 품질점수 기반 5/4/3등급으로 분류, 사용자가 "더 보기/덜 보기"로 정리 범위를 직접 조절 |
| 갤러리형 | 빠른 대량정리 불편, 삭제마다 확인 필요 | 그리드에서 가로 스와이프로 확인 없이 즉시 삭제, 드래그 연속 삭제, 삭제대기함으로 복구 가능 |

**핵심 기능 상세**

| 핵심 기능 | 구현 방식 |
|----------|----------|
| 유사사진 얼굴 크롭·비교 | 이미지 특징 벡터 간 유사도 비교로 연속 촬영 사진을 자동 그룹화. 그룹 내 얼굴 감지 → 임베딩(얼굴 고유 특징값) 추출 → 코사인 유사도(벡터 간 유사성 측정) 비교로 동일 인물을 자동 매칭. 인물별 얼굴을 크롭·확대하여 표정·눈감김 등 미세한 차이를 직관적으로 비교 |
| 품질점수 기반 3단계 자동정리 | 4단계 순차 필터: ① 메타데이터 필터(즐겨찾기·편집됨 등 보호) → ② 노출 분석(극단 어두움/밝음) → ③ 블러 분석(Laplacian Variance 기반 선명도 수치화) → ④ Safe Guard(Portrait 모드·선명한 얼굴 보호로 오탐 방지). 감지 결과를 품질 5/4/3등급으로 분류, 사용자가 정리 범위 조절 및 사진 선별 가능 |
| 그리드 스와이프 즉시 삭제 | 그리드(썸네일 목록)에서 가로 스와이프로 확인 없이 즉시 삭제 및 복구. 드래그로 여러 장 연속 삭제 가능. 햅틱 피드백으로 삭제를 직관적으로 인지하며, 삭제된 사진은 그리드에서 어둡게 표시하여 시각적으로 구분 |
| 고성능 이미지 파이프라인 | 우선순위 큐 기반 비동기 썸네일 로딩, 인접 행 프리히트(미리 로드), 스크롤 중 저화질 자동 전환, LRU 캐싱(최근 사용 데이터 우선 보관)을 조합하여 5만 장 이상에서도 120fps 스크롤 유지 |
| 삭제대기함 (안전장치) | 삭제된 사진을 앱 내 임시 저장소에 보관하여 언제든 개별 복구 가능. 삭제대기함 전용 뷰어에서 사진을 다시 확인한 뒤 복구 또는 최종 삭제를 선택할 수 있으며, 확인 팝업 없는 즉시 삭제 UX를 안심하고 사용할 수 있게 하는 안전장치 |

인물 매칭 기반 유사사진 인식 하이브리드 엔진, 다중 안전장치 기반 4단계 품질 분석 엔진, 대용량 환경 적응형 이미지 파이프라인 등 핵심 기술 독자 개발 및 적용.

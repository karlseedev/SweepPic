## `1. 문제 인식 (Problem)`

### `1. 창업 아이템의 필요성`
**사진 보관량 증가와 정리 수요**

- 전 세계 연간 약 2조 장의 사진이 촬영되고 있으며 꾸준한 증가 추세¹⁾
- 글로벌 Photo & Video 앱 시장 연평균 8~9% 성장 중¹⁰⁾
- 설문조사 결과 사용자는 자동 삭제를 신뢰하지 않으며, 직접 보고 판단하고 싶어 함⁴⁾
- 다양한 정리 앱이 출시되었으나 30일 리텐션 1.5%(전 카테고리 최하위)로 사용자 만족도가 낮음⁵⁾

**기존 솔루션의 한계**

| 유형 | 한계 | 대표앱 |
|------|------|--------|
| 수동정리형 | 한 장씩 넘기며 판단하여 대량 정리가 어려움. 유사 사진 간 표정 등 미세한 차이 비교가 불편 | Slidebox, SwipeWipe |
| 자동정리형 | AI가 획일적 기준으로 저품질 판정. 정리 수준 선택 불가능 | Cleanup, Cleaner Guru |
| 기타(갤러리 형) | 시간순·앨범별 탐색에 최적화, 정리 전용 기능이 미흡. 삭제 시마다 개별 선택과 확인이 필요 | HashPhotos, MyPics |

**AI 스마트 사진 정리 솔루션 'SweepPic'**

| 항목 | 내용 |
|------|------|
| 제품명 | SweepPic |
| 개요 | 빠르고 직관적인 정리 기능을 제공하는 사진 관리 앱 |

| 핵심 가치 | ① 얼굴 크롭·비교를 통한 유사사진 정리 편의성 극대화 ② 품질점수 기반 3단계 옵션으로 보다 정밀한 자동 정리 기능 제공 ③ 갤러리 인터페이스와 섬네일 즉시 삭제로 차별화된 정리 경험 제공 |


**SweepPic만의 상세 핵심 기능**

기존 시장에 없는 3가지 특화 기능을 개발하여, 빠르고 직관적인 솔루션 제공

| 특화 기능 | 상세 |
|----------|------|
| 유사사진 얼굴 크롭·비교 | • 이미지 특징 벡터 간 유사도 비교로 연속 촬영 사진을 자동 그룹화 • 인물별 얼굴을 크롭·확대하여 표정·눈감김 등의 차이를 직관적으로 비교 |
| 품질점수 기반 3단계 자동정리 | • 4단계 순차 필터로 흐림·과노출 등의 저품질 사진을 분류 • 감지 결과를 품질 5/4/3등급으로 분류 • 사용자가 정리 범위 선택 기능 및 편리한 사진 선별 |
| 그리드 스와이프 즉시 삭제 | • 그리드(섬네일 목록)에서 섬네일 가로 스와이프로 확인 팝업 없이 즉시 삭제 및 복구. 드래그로 여러 장 연속 삭제 가능 • 삭제된 사진은 그리드에서 어두운 빨간색으로 표시하여 시각적으로 구분 |

**경쟁력 확보 전략**

하이브리드 엔진 및 품질 분석 엔진 등 각 기능별 핵심 기술 독자 개발 및 적용

| 기술 경쟁력 | 상세 |
|------------|------|
| 얼굴 인식·비교 기술 | • 얼굴 고유 특징값 추출, 벡터 간 코사인 유사도 비교로 인물 자동 매칭 • 위치 기반 1차 매칭 + 임베딩 값 2차 검증의 하이브리드 방식으로 동일 인물 인식 정확도 확보 |

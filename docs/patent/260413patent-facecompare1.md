## 발명제안서 초안

### 1. 발명의 명칭

- 국문: 유사 사진 그룹 내부에서 인물별 얼굴 비교를 통해 원본 사진 정리를 수행하는 방법 및 사용자 인터페이스
- 영문: Method and User Interface for Organizing Original Photos Through Person-Specific Face Comparison Within a Similar Photo Group

### 2. 기술분야

본 발명은 사진 관리용 사용자 인터페이스 및 컴퓨터구현발명에 관한 것으로, 보다 구체적으로는 유사한 장면의 복수 사진을 그룹화한 후, 그룹 내부에서 동일 인물별 얼굴 비교를 수행하여 원본 사진의 선택 또는 삭제 판단을 지원하는 기술에 관한 것입니다.

### 3. 배경기술

기존 사진 관리 기술은 대체로 다음 두 방향으로 발전해 왔습니다.

- 전체 라이브러리 수준에서 얼굴을 인식하여 사람별 앨범이나 태그를 생성하는 방식
- 유사하거나 연속 촬영된 사진을 하나의 스택 또는 그룹으로 묶고 대표 사진을 선택하는 방식

그러나 실제 사용자의 사진 정리 상황에서는, 같은 장면을 여러 장 촬영한 사진 묶음 안에서 특정 인물의 표정, 눈뜸 여부, 시선, 흔들림 상태 등을 비교하여 더 적절한 사진을 남기고 나머지를 정리하고자 하는 수요가 자주 발생합니다.

기존 방식은 사진 전체를 훑어보거나, 시스템이 자동으로 대표 사진을 제안하는 수준에 머무는 경우가 많아, 복수 인물이 포함된 유사 사진 그룹에서 `인물별 상태 비교를 바탕으로 원본 사진을 정리`하는 작업에는 비효율적입니다.

### 4. 종래기술의 문제점

기존 기술은 다음과 같은 한계를 가집니다.

1. 전체 라이브러리 단위의 사람 분류는 인물 아카이빙에는 적합하지만, 같은 장면의 유사 사진 정리에는 직접적이지 않습니다.
2. 유사 사진 그룹화 기술은 사진 묶음을 만들 수 있으나, 복수 인물이 있는 경우 어떤 사진을 남길지 판단하는 기준을 충분히 제공하지 못합니다.
3. 얼굴 클러스터링 또는 사람 브라우징 기술은 얼굴 자체의 식별이나 태깅에는 유용하지만, 그 결과를 원본 사진 정리 인터페이스로 직접 연결하지는 않습니다.
4. 자동 대표컷 선택 기술은 시스템 판단에 의존하는 경향이 있어, 사용자가 사람별 얼굴 상태를 직접 비교하면서 정리하고자 하는 요구를 충족하기 어렵습니다.

### 5. 해결하고자 하는 과제

본 발명은 다음 과제를 해결하고자 합니다.

- 유사 사진 그룹 내부에서 복수 인물의 상태를 빠르게 비교할 수 있도록 할 것
- 사진 전체가 아니라 동일 인물의 얼굴 단위로 비교 기준을 제공할 것
- 얼굴 비교 결과를 원본 사진 선택 또는 삭제 판단으로 직접 연결할 것
- 시스템 자동 선정이 아니라 사용자 판단을 효율적으로 보조하는 정리 인터페이스를 제공할 것

### 6. 해결수단의 요지

본 발명은 유사 사진 그룹을 먼저 형성한 후, 해당 그룹 내부에 포함된 각 사진에서 얼굴을 검출하고 동일 인물에 대응되는 얼굴들에 인물 식별 인덱스를 부여합니다. 이후 동일 인물에 대응되는 얼굴 crop들을 사용자에게 비교 가능하게 제시하고, 사용자는 상기 얼굴 비교 결과를 바탕으로 원본 사진 단위의 선택, 제외 또는 삭제를 수행합니다.

여기서 본 발명의 핵심은 다음과 같습니다.

1. 인물 매칭의 범위를 전체 라이브러리가 아니라 `유사 사진 그룹 내부`로 제한하는 점
2. 비교 단위는 `얼굴 crop`이지만, 실제 처리 단위는 `원본 사진`인 점
3. 시스템이 대표 사진을 자동 선정하는 것이 아니라, 사용자에게 인물별 비교 기준을 제공하여 사용자 스스로 정리 판단을 하도록 하는 점

### 7. 발명의 핵심 구성

본 발명의 일 실시예는 다음 구성요소를 포함할 수 있습니다.

- 유사 사진들을 그룹화하는 그룹 형성부
- 그룹 내부 사진들에서 얼굴을 검출하는 얼굴 검출부
- 그룹 내부에 한정하여 동일 인물 여부를 판정하는 인물 매칭부
- 동일 인물별 얼굴 crop들을 분리하여 제공하는 비교 인터페이스 구성부
- 얼굴 비교 결과를 바탕으로 원본 사진 단위의 선택 또는 삭제를 반영하는 정리 처리부

### 8. 발명의 작동 방식

본 발명의 일 실시예에 따른 작동 방식은 다음과 같습니다.

1. 시스템은 복수의 사진 중 유사한 장면 또는 연속 촬영 조건을 만족하는 사진들을 하나의 유사 사진 그룹으로 형성합니다.
2. 시스템은 상기 유사 사진 그룹 내부의 각 사진에서 하나 이상의 얼굴을 검출합니다.
3. 시스템은 상기 얼굴들에 대하여 그룹 내부 범위에서만 동일 인물 여부를 판정하여, 동일 인물에 대응되는 얼굴들에 공통 인물 식별 정보를 부여합니다.
4. 시스템은 동일 인물에 대응되는 얼굴 crop들을 사용자 인터페이스상에서 비교 가능한 형태로 제시합니다.
5. 사용자는 동일 인물의 얼굴 상태를 비교하여, 어떤 원본 사진을 유지하고 어떤 원본 사진을 정리할지 판단합니다.
6. 시스템은 상기 판단 결과를 얼굴 crop이 아닌 원본 사진 단위의 선택, 제외, 삭제 또는 삭제대기함 이동으로 반영합니다.

### 9. 핵심 차별 포인트

#### 9.1 파이프라인 범위 및 목적의 차이

기존 사람 인식 기술은 일반적으로 전체 라이브러리에서 인물을 분류하여 사람 앨범을 생성하는 방향에 가깝습니다. 반면 본 발명은 `유사 사진 그룹 내부`라는 제한된 범위 안에서만 인물 매칭을 수행하며, 목적 또한 사람 식별 자체가 아니라 `사진 정리 판단`에 있습니다.

#### 9.2 비교 단위와 처리 단위의 분리

본 발명에서 사용자는 얼굴 crop을 비교 기준으로 활용하지만, 실제 선택 또는 삭제는 원본 사진에 적용됩니다. 즉, `비교는 얼굴 단위로 수행되되, 정리는 사진 단위로 수행되는 구조`를 가집니다.

#### 9.3 사용자 판단 보조형 인터페이스

기존 기술 중 일부는 시스템이 품질 점수나 표정 점수에 따라 대표 사진을 자동 선택합니다. 반면 본 발명은 시스템이 동일 인물별 얼굴 비교 재료를 구성해 제공하고, 사용자가 직접 더 적절한 사진을 판단하여 정리하도록 지원합니다.

#### 9.4 인물별 비교 단위의 분리 제공

유사 사진 그룹에 복수 인물이 포함된 경우에도, 본 발명은 동일 인물별로 분리된 비교 단위를 제공함으로써, 사용자가 한 사람씩 상태를 비교하며 정리 판단을 내릴 수 있도록 합니다.

### 10. 발명의 효과

본 발명에 의하면 다음과 같은 효과를 얻을 수 있습니다.

1. 같은 장면의 여러 사진에서 사람별 표정이나 상태를 빠르게 비교할 수 있습니다.
2. 복수 인물이 있는 경우에도 사진 전체를 반복 확인하지 않고 인물 중심으로 정리 판단을 할 수 있습니다.
3. 자동 대표컷 추천보다 사용자의 의도를 더 정확히 반영할 수 있습니다.
4. 얼굴 비교를 원본 사진 정리와 직접 연결함으로써 사진 정리 작업의 효율을 높일 수 있습니다.

### 11. 권리화 포인트 제안

#### 11.1 독립항 방향

유사 사진 그룹 내부에서 동일 인물에 대응되는 얼굴 crop들을 생성 및 분리 제시하고, 사용자가 상기 얼굴 crop 비교 결과를 기초로 원본 사진 단위의 선택 또는 삭제를 수행하도록 하는 방법

#### 11.2 종속항 방향

- 인물 매칭을 전체 라이브러리가 아니라 그룹 내부에 한정하는 점
- 복수 인물이 포함된 경우 인물별로 분리된 비교 단위를 순차 제공하는 점
- 얼굴 crop 비교 결과가 삭제대기함 이동 또는 원본 사진 제외 처리로 연결되는 점
- 동일 인물의 얼굴 상태 비교 기준이 표정, 눈뜸 여부, 시선, 흔들림 또는 얼굴 품질을 포함하는 점

### 12. 요약

본 발명은 유사 사진 그룹 내부에서 동일 인물별 얼굴 crop을 분리하여 제시하고, 사용자가 상기 얼굴 비교 결과를 바탕으로 원본 사진 단위의 선택 또는 삭제를 수행하도록 하는 사진 정리 방법 및 사용자 인터페이스에 관한 것입니다.

즉, 본 발명의 본질은 `얼굴을 인식한다`는 점보다, `얼굴 비교를 사진 정리의 판단 인터페이스로 전환한다`는 데 있습니다.

## 선행기술 참고 링크

아래 문헌들은 `유사사진 그룹 내부에서 동일 인물의 얼굴을 비교하여 사진 정리로 연결하는 흐름`과 관련성이 비교적 높은 선행기술입니다.

- [US8189880B2 - Interactive photo annotation based on face clustering](https://patents.google.com/patent/US8189880B2/en)
  - 얼굴 유사도에 따라 사진 컬렉션을 복수의 face group으로 클러스터링하고, UI에서 얼굴 그룹을 검토/병합/정리하는 흐름을 포함함
  - `얼굴 그룹을 사용자 인터페이스에서 직접 검토한다`는 점에서 가장 가까운 문헌 중 하나

- [US8531478B2 - Method of browsing photos based on people](https://patents.google.com/patent/US8531478B2/en)
  - 앨범의 사진들과 함께 사람별 대표 얼굴을 표시하고, 특정 사람을 선택하면 해당 인물의 얼굴 영역들을 별도로 보여주는 browsing UI를 포함함
  - `사진 목록과 사람별 얼굴 표시를 결합한 UI`라는 점에서 참고 가치가 높음

- [US10430456B2 - Automatic grouping based handling of similar photos](https://patents.google.com/patent/US10430456B2/en)
  - 유사한 장면의 사진들을 facial features, background composition, color composition 등을 바탕으로 그룹화하고, 그룹 단위 또는 사진 단위로 처리하는 흐름을 포함함
  - `유사사진 그룹화 후 처리`라는 전단계 측면에서 직접 관련됨

- [US9858296B2 - Representative image selection for image management using face recognition](https://patents.google.com/patent/US9858296B2/en)
  - 특정 인물의 얼굴 표정 데이터를 추출하여 그룹 내 이미지의 quality parameter를 비교하고 대표 이미지를 선택하는 내용을 포함함
  - `표정/얼굴 상태를 비교해 더 적절한 사진을 고른다`는 측면에서 관련성이 있음

- [US8837859B2 - System and method for image combination using corresponding regions in images](https://patents.google.com/patent/US8837859B2/en)
  - 동일 장면의 여러 사진에서 대응 영역을 비교하고 선택하는 흐름을 포함하며, 얼굴/표정 품질 판단과 연관된 설명이 있음
  - `같은 장면의 여러 사진에서 사람별 상태를 비교한다`는 문제의식과 닿아 있음

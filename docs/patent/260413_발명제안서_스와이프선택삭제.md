# 발명제안서

---

## 1. 발명의 명칭

**[국문]** 그리드 형태의 콘텐츠 목록에서 선택 모드 전환 없이 스와이프 제스처를 이용한 항목 선택 및 작업 실행 방법과 이를 수행하는 전자 장치

**[영문]** Method for Selecting Items and Executing Actions Using Swipe Gestures Without Selection Mode Switching in a Grid-Based Content List, and Electronic Device Performing the Same

---

## 2. 기술분야

본 발명은 터치스크린을 구비한 전자 장치에서 그리드(격자) 형태로 배열된 콘텐츠 항목에 대해, 별도의 선택 모드 전환 없이 스와이프 제스처를 이용하여 항목을 선택하거나 소정의 작업(삭제, 이동, 분류 등)을 수행하되, 상태 변경 이후에도 탭 입력에 의한 상세 열람이 항상 유지되는 비모달(non-modal) 인터페이스 구조를 포함하는 사용자 인터페이스 방법 및 장치에 관한 것이다.

---

## 3. 발명의 배경이 되는 기술

### 3.1 종래기술의 설명

터치스크린 기반 모바일 장치에서 사진, 파일, 연락처 등의 콘텐츠를 그리드(격자) 형태로 표시하고 관리하는 것은 널리 사용되는 인터페이스 방식이다.

종래기술에서 그리드 형태의 콘텐츠에 대한 선택 및 작업 수행은 주로 다음과 같은 방식으로 이루어진다:

**(1) 길게 누르기(Long Press) 기반 선택 모드 진입**

사용자가 특정 항목을 길게 눌러 선택 모드로 진입한 후, 개별 항목을 하나씩 탭하여 선택하는 방식이다. 이 방식은 모달(modal) 인터페이스로서, 선택 모드에 진입하면 탭 제스처의 의미가 "콘텐츠 열람"에서 "항목 선택"으로 변경된다. 따라서 선택 모드 중에는 특정 항목을 탭하여 상세 뷰어로 진입할 수 없으며, 상세 뷰어로 진입하려면 먼저 선택 모드를 해제해야 한다. 또한 다수의 항목을 선택할 때 각 항목을 개별적으로 탭해야 하므로 조작 횟수가 많고, 선택 모드 진입을 위한 별도의 대기 시간(길게 누르기)이 필요하다.

**(2) 리스트(목록) 기반 스와이프 삭제**

이메일, 메시지 등의 리스트 형태 인터페이스에서 개별 항목을 수평으로 스와이프하여 삭제하거나 상황별 메뉴를 노출하는 방식이다. 이 방식은 리스트(1열) 형태에 특화되어 있어, 그리드(다열) 형태에서는 적용이 어렵고, 한 번에 하나의 항목에 대해서만 동작한다.

**(3) 전체화면 뷰어에서의 스와이프 삭제**

사진 등의 콘텐츠를 전체화면으로 확대하여 보는 상태에서, 위 또는 아래 방향으로 스와이프하여 해당 항목을 삭제하는 방식이다. 이 방식은 한 번에 하나의 항목만 처리할 수 있고, 전체화면 뷰어로 진입해야 하는 별도의 단계가 필요하다.

**(4) 두 손가락 드래그 선택**

두 손가락으로 그리드 위를 드래그하여 범위 내 항목을 선택하는 방식이다. 이 방식은 두 손가락을 동시에 사용해야 하므로 한 손 조작이 어렵고, 선택 후 별도의 작업 버튼(삭제, 이동 등)을 추가로 탭해야 한다.

### 3.2 종래기술의 문제점

상기한 종래기술들은 다음과 같은 문제점을 갖는다:

- **선택 모드 진입 시 탭에 의한 상세 열람이 차단되는 문제(핵심 문제)**: 종래의 선택 모드 방식에서는, 선택 모드에 진입하면 동일한 탭 제스처가 "콘텐츠 열람"이 아닌 "항목 선택"으로 의미가 변경된다. 이로 인해 사용자가 선택 작업 도중 특정 항목의 상세 내용을 확인하고자 할 때, 반드시 선택 모드를 먼저 해제한 후 다시 탭해야 하는 불필요한 단계가 발생한다. 상세 확인 후 다시 선택 작업을 이어가려면 선택 모드에 재진입해야 하므로, 열람과 선택을 교차하는 작업 흐름에서 매 교차 시마다 최소 2회의 모드 전환이 필요하여 심각한 비효율이 발생한다.
- **조작 단계의 과다**: 선택 모드 진입 → 개별 선택 → 작업 버튼 탭 등 다단계 조작이 필요하여 사용자의 작업 효율이 낮다.
- **단일 항목 한정**: 리스트 스와이프, 전체화면 스와이프 방식은 한 번의 제스처로 하나의 항목만 처리 가능하다.
- **모드 전환의 단절감**: 일반 모드와 선택 모드 사이의 전환이 명시적으로 이루어져야 하여 사용자 경험의 연속성이 끊어진다.
- **시각적 피드백 부족**: 제스처 진행 중 작업의 적용 범위나 진행도를 직관적으로 확인하기 어렵다.
- **확인 다이얼로그의 작업 중단**: 삭제 등 비가역적 작업 수행 시 확인 다이얼로그가 표시되어 연속적인 작업 흐름이 중단된다.

## 5. 발명의 내용

### 5.1 해결하고자 하는 과제

본 발명은 상기한 종래기술의 문제점을 해결하기 위한 것으로, 다음과 같은 기술적 과제를 해결하고자 한다:

**(과제 1 — 핵심)** 그리드 형태의 콘텐츠 목록에서, 별도의 선택 모드 전환 없이 스와이프 제스처에 의한 항목 선택 및 작업 수행이 가능하면서도, 상태 변경 이후에도 탭 입력에 의한 상세 열람이 항상 유지되는 인터페이스 구조를 제공한다. 이에 의해, 사용자가 스와이프로 항목에 대한 작업을 수행한 직후에도 언제든지 다른 항목을 탭하여 상세 뷰어로 진입할 수 있고, 상세 뷰어에서 복귀한 후 별도의 모드 재진입 없이 스와이프 작업을 이어갈 수 있도록 한다.

**(과제 2)** 그리드 형태의 콘텐츠 목록에서 별도의 모드 전환 없이 한 손가락의 스와이프 제스처만으로 항목의 선택 및 작업 수행을 즉시 실행할 수 있는 방법을 제공한다.

**(과제 3)** 단일 항목에 대한 스와이프 제스처가 진행 중에 자연스럽게 다중 항목 선택으로 확장될 수 있는 연속적 인터랙션 방법을 제공한다.

**(과제 4)** 스와이프 제스처의 진행도를 실시간으로 시각적 피드백(오버레이 커튼 효과)으로 제공하여 사용자가 작업의 적용 여부를 제스처 도중에 판단하고 제어할 수 있는 방법을 제공한다.

**(과제 5)** 확인 다이얼로그 없이 즉시 작업을 수행하되, 실행 취소(Undo) 메커니즘을 통해 안전성을 확보하는 방법을 제공한다.

### 5.2 과제의 해결 수단

상기 과제를 해결하기 위하여, 본 발명은 다음과 같은 구성을 포함하는 방법 및 장치를 제공한다:

#### 5.2.0 선택 모드 전환이 불필요한 제스처 역할 고정 구조 (핵심 구성)

본 발명의 그리드 인터페이스에서, 각 제스처 유형에 대해 고정된 역할이 할당되며, 이 역할은 항목의 상태 변경(선택, 삭제 등) 전후를 포함한 어떠한 시점에서도 변경되지 않는다:

- **탭(tap) 제스처**: 항상 해당 항목의 상세 뷰어(전체화면 보기 등)를 실행한다.
- **수평 스와이프(swipe) 제스처**: 항상 해당 항목에 대한 선택 또는 소정의 작업(삭제, 이동 등)을 수행한다.
- **수직 스와이프 제스처**: 항상 그리드의 스크롤을 수행한다.

상기 제스처 역할 고정에 의해, 사용자는 별도의 선택 모드에 진입하거나 해제할 필요 없이, 어느 시점에서든 탭으로 콘텐츠를 열람하고 스와이프로 항목을 선택/작업할 수 있다. 즉, 스와이프로 일부 항목에 대한 작업을 수행한 직후에도, 다른 항목을 탭하면 즉시 상세 뷰어로 진입할 수 있으며, 상세 뷰어에서 복귀한 후에도 추가적인 스와이프 작업을 별도의 모드 전환 없이 바로 수행할 수 있다.

이는 별도의 선택 모드 전환 없이, 상태 변경 이후에도 탭 입력에 의한 상세 열람이 유지되는 비모달(non-modal) 인터페이스 구조로서, 종래의 모달 선택 모드 방식과 근본적으로 구별되는 본 발명의 핵심 특징이다.

#### 5.2.1 스와이프 제스처 인식 및 방향 판별 단계

터치스크린에 표시된 그리드 형태의 콘텐츠 목록에서, 사용자의 한 손가락 터치 입력에 의한 팬(pan) 제스처를 감지하고, 소정의 최소 이동 거리(예: 10포인트) 이상 이동 후 이동 방향의 각도를 산출하여, 산출된 각도가 소정의 각도 임계값(예: 수평 기준 25도) 이내인 경우에만 스와이프 제스처로 인식하고, 상기 각도 임계값을 초과하는 경우에는 스크롤 제스처로 판별하여 스와이프 제스처 인식을 취소한다.

#### 5.2.2 단일 항목 스와이프 모드

스와이프 제스처가 인식되면, 제스처가 시작된 위치에 대응하는 그리드 셀(이하 "대상 셀")을 특정하고, 제스처의 수평 이동 거리에 기반하여 대상 셀의 폭 대비 진행도(progress)를 산출한다. 산출된 진행도에 따라 대상 셀 위에 소정의 오버레이를 제스처 방향으로부터 점진적으로 표시(이하 "커튼 효과")하여, 사용자에게 작업 적용의 진행 상태를 실시간으로 시각적으로 피드백한다.

#### 5.2.3 단일-다중 자동 전환 단계

상기 단일 항목 스와이프 모드에서 제스처가 진행되는 중에, 사용자의 손가락이 대상 셀을 벗어나 인접한 다른 셀의 영역에 도달하는 것이 감지되면, 별도의 모드 전환 조작 없이 자동으로 다중 항목 선택 모드로 전환된다. 이때, 최초에 스와이프가 시작된 셀(이하 "앵커 셀")의 위치 정보(행, 열)가 기준점으로 설정된다.

#### 5.2.4 다중 항목 범위 산출 단계

다중 항목 선택 모드에서, 앵커 셀의 행/열 위치와 현재 손가락이 위치하는 셀의 행/열 위치를 비교하여 선택 범위를 산출한다:

- **(a) 동일 행 선택**: 앵커 셀과 현재 셀이 동일 행에 위치하는 경우, 앵커 셀의 열부터 현재 셀의 열까지의 범위에 해당하는 셀들이 선택된다.
- **(b) 다중 행 선택**: 앵커 셀과 현재 셀이 서로 다른 행에 위치하는 경우, 앵커 셀의 행부터 현재 셀의 행까지의 모든 행에 대해 해당 행의 전체 열에 해당하는 셀들이 사각형 범위로 선택된다.

손가락의 이동에 따라 선택 범위가 실시간으로 갱신되며, 이전에 선택 범위에 포함되었다가 벗어난 셀은 원래 상태로 즉시 복귀한다.

#### 5.2.5 다중 항목 시각 피드백 단계

다중 항목 선택 모드에서, 선택 범위에 새로 포함된 셀에 대해서는 소정의 오버레이가 즉시 적용되고, 현재 손가락이 위치하는 셀(이하 "커튼 셀")에 대해서는 상기 단일 항목 모드와 동일한 커튼 효과가 적용된다. 이로써 사용자는 손가락의 수평 위치에 따라 커튼 셀에 대한 작업 적용 여부를 정밀하게 제어할 수 있다.

#### 5.2.6 제스처 확정 판단 단계

사용자가 손가락을 터치스크린에서 떼는 시점에, 다음 두 가지 조건 중 하나 이상을 충족하는 경우 작업 수행을 확정한다:

- **(조건 A) 거리 조건**: 스와이프의 수평 이동 거리가 대상 셀 폭의 소정 비율(예: 50%) 이상인 경우
- **(조건 B) 속도 조건**: 스와이프의 수평 이동 속도가 소정의 속도 임계값(예: 800포인트/초) 이상인 경우

상기 조건을 모두 충족하지 않는 경우에는 제스처가 취소된 것으로 판단하여, 대상 셀(들)을 원래 상태로 복귀시키는 취소 애니메이션을 실행한다.

#### 5.2.7 작업 수행 및 실행 취소 단계

제스처가 확정되면, 확인 다이얼로그를 표시하지 않고 선택된 항목에 대해 소정의 작업(삭제, 선택, 이동, 분류 등)을 즉시 수행한다. 작업 수행과 동시에 실행 취소(Undo) 인터페이스를 제공하여, 사용자가 소정의 시간 내에 작업을 되돌릴 수 있도록 한다.

#### 5.2.8 상태 인지 양방향 동작

대상 항목의 현재 상태에 따라 동일한 스와이프 제스처에 대해 상반된 작업을 수행한다. 예를 들어, 미삭제 상태의 항목에 대한 스와이프는 삭제 작업을 수행하고, 이미 삭제된 상태의 항목에 대한 스와이프는 복구 작업을 수행한다. 이때, 오버레이의 방향이 반전되어 시각적으로 구별된다:

- **정방향 작업(예: 삭제)**: 오버레이가 스와이프 시작점으로부터 진행 방향으로 채워진다.
- **역방향 작업(예: 복구)**: 오버레이가 스와이프 진행 방향으로 걷혀 나간다.

### 5.3 발명의 효과

본 발명에 의하면, 다음과 같은 효과를 얻을 수 있다:

**(효과 1 — 핵심) 상태 변경 이후에도 탭에 의한 상세 열람이 유지**: 스와이프에 의해 항목의 상태가 변경(삭제, 선택 등)된 이후에도, 탭 입력에 의한 상세 뷰어 진입이 항상 유지되므로, 사용자는 스와이프 작업 도중에도 언제든지 다른 항목을 탭하여 상세 내용을 확인할 수 있다. 상세 뷰어에서 복귀한 후에도 별도의 모드 재진입 없이 스와이프 작업을 이어갈 수 있어, 종래의 모달 선택 모드에서 필수적이었던 "선택 모드 해제 → 열람 → 선택 모드 재진입"의 반복적 전환이 완전히 제거된다.

**(효과 2) 조작 단계 최소화**: 별도의 모드 진입, 개별 항목 탭, 작업 버튼 탭 등의 다단계 조작 없이, 한 번의 연속된 스와이프 제스처만으로 항목 선택부터 작업 수행까지 완료할 수 있어 사용자의 작업 효율이 크게 향상된다.

**(효과 3) 연속적 사용자 경험**: 단일 항목 조작에서 다중 항목 조작으로 전환될 때 별도의 모드 전환 조작이 불필요하므로, 사용자 경험의 연속성이 유지된다.

**(효과 4) 직관적 시각 피드백**: 커튼 효과에 의해 스와이프 진행도가 실시간으로 시각화되므로, 사용자가 제스처 도중에 작업의 적용 범위와 정도를 직관적으로 파악하고, 손가락을 떼는 시점과 위치를 조절하여 작업의 확정 또는 취소를 정밀하게 제어할 수 있다.

**(효과 5) 단일/다중 범위의 유연한 제어**: 손가락이 동일 행 내에 머무르면 열 단위의 정밀한 선택이 가능하고, 다른 행으로 이동하면 행 단위의 대량 선택이 가능하여, 사용자가 의도하는 범위에 따라 유연하게 선택 범위를 조절할 수 있다.

**(효과 6) 스크롤과의 자연스러운 공존**: 각도 기반 판별에 의해 스와이프 제스처와 스크롤 제스처가 충돌 없이 공존하므로, 기존의 스크롤 동작에 영향을 주지 않으면서도 추가적인 기능을 제공할 수 있다.

**(효과 7) 작업 흐름의 연속성**: 확인 다이얼로그 없이 즉시 작업이 수행되므로 연속적인 작업 흐름이 중단되지 않으며, 실행 취소 메커니즘에 의해 오조작에 대한 안전성도 확보된다.

**(효과 8) 상태 인지 양방향 동작**: 동일한 제스처로 정방향 작업(예: 삭제)과 역방향 작업(예: 복구)을 모두 수행할 수 있으므로, 사용자가 추가적인 조작 방법을 학습할 필요 없이 직관적으로 양방향 작업을 수행할 수 있다.

## 6. 도면의 간단한 설명

**[도 1]** 본 발명의 전체 처리 흐름을 나타내는 순서도

**[도 2]** 그리드 형태의 콘텐츠 목록에서 단일 항목 스와이프 제스처의 진행 과정을 나타내는 화면 구성도
- (a) 스와이프 시작 전 상태
- (b) 스와이프 진행 중 (커튼 효과 30% 진행)
- (c) 스와이프 진행 중 (커튼 효과 70% 진행)
- (d) 스와이프 확정 후 상태

**[도 3]** 단일 항목 모드에서 다중 항목 모드로 자동 전환되는 과정을 나타내는 화면 구성도
- (a) 단일 셀에서 스와이프 시작
- (b) 손가락이 인접 셀에 도달하여 다중 모드 진입
- (c) 동일 행 내 범위 선택
- (d) 다른 행으로 이동 시 사각형 범위 선택

**[도 4]** 다중 항목 선택 모드에서의 범위 산출 방법을 나타내는 도면
- (a) 동일 행 선택: 앵커 열 ~ 현재 열 범위
- (b) 다중 행 선택: 앵커 행 ~ 현재 행의 전체 열 사각형 범위

**[도 5]** 커튼 효과의 시각적 피드백 메커니즘을 나타내는 도면
- (a) 정방향 작업(삭제): 오버레이가 스와이프 방향으로 채워지는 모습
- (b) 역방향 작업(복구): 오버레이가 스와이프 방향으로 걷히는 모습

**[도 6]** 제스처 확정/취소 판단 조건을 나타내는 도면
- (a) 거리 조건에 의한 확정
- (b) 속도 조건에 의한 확정
- (c) 조건 미충족 시 취소 애니메이션

**[도 7]** 스와이프 제스처와 스크롤 제스처의 각도 기반 판별 과정을 나타내는 도면

**[도 8]** 상태 인지 양방향 동작의 작동 과정을 나타내는 상태 전이도

**[도 9]** 종래기술과 본 발명의 열람-작업 교차 흐름 비교를 나타내는 도면
- (a) 종래기술: 선택 모드 진입 → 탭(선택) → 모드 해제 → 탭(열람) → 모드 재진입 → 탭(선택) — 매 교차 시 2회 모드 전환 필요
- (b) 본 발명: 스와이프(작업) → 탭(열람) → 스와이프(작업) — 선택 모드 전환 없이 탭에 의한 상세 열람이 항상 유지됨

## 7. 발명의 실시를 위한 구체적인 내용

### 실시예 1: 사진 관리 앱에서의 스와이프 삭제

사용자의 모바일 장치에 저장된 사진들이 3열 그리드 형태의 썸네일로 표시된다. 사용자가 삭제하고자 하는 사진의 썸네일 위에서 한 손가락으로 수평 방향(좌 또는 우)으로 스와이프한다.

**(단계 S101)** 사용자의 터치 입력이 감지되고, 팬 제스처가 시작된다.

**(단계 S102)** 터치 이동 거리가 10포인트에 도달하면, 이동 방향의 각도를 산출한다. 수평 기준 25도 이내이므로 스와이프 제스처로 인식된다.

**(단계 S103)** 스와이프가 시작된 위치에 대응하는 그리드 셀(대상 셀)이 특정된다. 대상 셀의 현재 상태가 미삭제 상태이므로, 마룬색(어두운 적색) 오버레이가 준비된다.

**(단계 S104)** 사용자의 손가락이 수평으로 이동함에 따라, 대상 셀의 폭 대비 이동 거리의 비율(진행도)이 산출되고, 마룬색 오버레이가 스와이프 시작점으로부터 진행 방향으로 해당 진행도만큼 점진적으로 표시된다(커튼 효과).

**(단계 S105)** 사용자가 손가락을 떼는 시점에, 수평 이동 거리가 셀 폭의 50% 이상이므로 거리 조건이 충족되어 삭제가 확정된다.

**(단계 S106)** 확인 다이얼로그 없이 해당 사진이 삭제대기함으로 이동되고, 화면 하단에 실행 취소(Undo) 토스트가 표시된다.

### 실시예 2: 다중 사진 연속 삭제

사용자가 연속된 여러 장의 사진을 한 번의 제스처로 삭제하고자 한다.

**(단계 S201)** 실시예 1의 단계 S101~S103과 동일하게, 첫 번째 셀(앵커 셀)에서 스와이프가 시작된다. 앵커 셀은 그리드의 2행 1열에 위치한다.

**(단계 S202)** 스와이프 진행 중 사용자의 손가락이 앵커 셀을 벗어나 2행 2열의 셀 영역에 도달한다. 시스템은 자동으로 다중 항목 선택 모드로 전환한다.

**(단계 S203)** 앵커 셀(2행 1열)에는 커튼 효과가 완료(오버레이 전체 적용) 상태로 전환되고, 새로 진입한 2행 2열의 셀에 커튼 효과가 시작된다. 동일 행이므로 1열~2열 범위가 선택된다.

**(단계 S204)** 사용자의 손가락이 계속 이동하여 3행 영역에 도달한다. 앵커 행(2행)부터 현재 행(3행)까지의 전체 열(1~3열)이 사각형 범위로 선택되어, 총 6개의 셀이 선택 상태가 된다. 선택된 셀들에는 오버레이가 즉시 적용된다.

**(단계 S205)** 사용자가 손가락을 떼면, 선택된 6개 항목에 대해 일괄 삭제가 수행된다.

### 실시예 3: 삭제된 항목의 복구

삭제대기함 화면에서, 이미 삭제된 사진들이 반투명한 적색 오버레이가 적용된 상태로 그리드에 표시된다.

**(단계 S301)** 사용자가 복구하고자 하는 사진의 썸네일 위에서 수평으로 스와이프한다.

**(단계 S302)** 대상 셀의 현재 상태가 삭제 상태이므로, 시스템은 복구 작업으로 판단한다. 녹색 오버레이가 준비되고, 기존의 적색 오버레이가 스와이프 진행 방향으로 걷혀나가는 역방향 커튼 효과가 적용된다.

**(단계 S303)** 스와이프가 확정되면, 해당 사진이 삭제대기함에서 복구된다.

### 실시예 4: 콘텐츠 선택 (삭제 외 용도)

파일 관리 앱에서, 사용자가 그리드 형태로 표시된 파일들 중 특정 파일들을 선택하여 폴더 이동, 공유, 즐겨찾기 등의 작업을 수행하고자 한다.

**(단계 S401)** 사용자가 선택하고자 하는 파일의 썸네일 위에서 수평으로 스와이프한다.

**(단계 S402)** 대상 셀에 소정의 선택 표시(예: 체크마크 아이콘 및 하이라이트 오버레이)가 커튼 효과로 점진적으로 표시된다.

**(단계 S403)** 손가락이 인접 셀로 이동하면 다중 선택 모드로 전환되어, 범위 내 셀들이 일괄 선택된다.

**(단계 S404)** 스와이프가 확정되면, 선택된 항목들에 대해 화면 하단 등에 작업 메뉴(이동, 공유, 즐겨찾기 등)가 표시된다.

### 실시예 5: 스와이프 작업과 탭 열람의 교차 수행 (핵심 실시예)

사용자가 사진 그리드에서 불필요한 사진을 정리하면서, 중간중간 특정 사진의 상세 내용을 확인하고자 한다. 본 실시예는 별도의 선택 모드 전환 없이, 상태 변경 이후에도 탭 입력에 의한 상세 열람이 유지되는 본 발명의 핵심 특징을 나타낸다.

**(단계 S501)** 사용자가 그리드에서 첫 번째 사진을 수평 스와이프하여 삭제한다. 해당 셀에 마룬색 오버레이가 적용되고, 사진이 삭제대기함으로 이동된다.

**(단계 S502)** 사용자가 다음 사진을 삭제할지 판단하기 위해 해당 사진의 썸네일을 탭한다. **별도의 모드 해제 없이**, 즉시 전체화면 상세 뷰어가 실행되어 사진의 상세 내용(원본 해상도, 메타데이터 등)을 확인할 수 있다.

**(단계 S503)** 사용자가 상세 뷰어에서 그리드 화면으로 복귀한다.

**(단계 S504)** 사용자가 **별도의 모드 재진입 없이**, 바로 다른 사진을 수평 스와이프하여 삭제한다.

**(단계 S505)** 상기 S501~S504의 열람(탭)과 선택/작업(스와이프)의 교차가 무제한 반복 가능하며, 어떠한 시점에서도 모드 전환 조작이 필요하지 않다.

상기 실시예는 종래의 모달 선택 모드 방식에서는 불가능한 작업 흐름이다. 종래 방식에서는 상기 S502 단계에서 반드시 선택 모드를 먼저 해제해야 하고, S504 단계에서 선택 모드에 다시 진입해야 하므로, 최소 2회의 추가 조작이 매 교차 시마다 발생한다.

### 실시예 6: 미리보기 화면에서의 제외 처리

자동 정리 기능의 미리보기 화면에서, 시스템이 삭제를 권장하는 사진들이 그리드로 표시된다. 사용자가 삭제를 원하지 않는 사진을 스와이프하여 삭제 대상에서 제외한다.

**(단계 S501)** 사용자가 제외하고자 하는 사진의 썸네일 위에서 수평으로 스와이프한다.

**(단계 S502)** 소정의 제외 표시(예: 녹색 오버레이)가 커튼 효과로 표시된다.

**(단계 S503)** 스와이프가 확정되면, 해당 항목이 삭제 대상 목록에서 제외된다.

터치스크린을 구비한 전자 장치에서 그리드 형태로 배열된 복수의 콘텐츠 항목에 대해, 별도의 선택 모드 전환 없이 스와이프 제스처를 이용하여 항목을 선택하고 소정의 작업을 수행하는 방법에 있어서,

(a) 상기 그리드에서 탭 제스처에 대해서는 항상 해당 항목의 상세 뷰어를 실행하고, 수평 스와이프 제스처에 대해서는 항상 해당 항목에 대한 선택 또는 소정의 작업을 수행하도록, 각 제스처 유형에 고정된 역할을 할당하되, 상기 스와이프 제스처에 의한 항목의 상태 변경 이후에도 상기 탭 제스처에 의한 상세 뷰어 실행이 유지되도록 구성하는 단계;

(b) 상기 그리드의 소정의 셀 위에서 사용자의 한 손가락에 의한 팬 제스처를 감지하고, 소정의 최소 이동 거리 이상 이동 후 이동 방향의 각도를 산출하여, 상기 각도가 소정의 임계값 이내인 경우 수평 스와이프 제스처로 인식하는 단계;

(c) 상기 스와이프 제스처가 인식된 셀(대상 셀)에 대해, 제스처의 수평 이동 거리에 기반한 진행도를 산출하고, 상기 진행도에 따라 상기 대상 셀 위에 오버레이를 점진적으로 표시하는 커튼 효과를 적용하는 단계;

(d) 상기 스와이프 제스처의 진행 중에 사용자의 접촉점이 상기 대상 셀의 영역을 벗어나 인접 셀의 영역에 도달하는 것이 감지되면, 별도의 모드 전환 조작 없이 자동으로 다중 항목 선택 모드로 전환하는 단계;

(e) 상기 다중 항목 선택 모드에서, 최초 스와이프가 시작된 앵커 셀의 위치와 현재 접촉점이 위치하는 셀의 위치에 기반하여 선택 범위를 산출하고, 상기 선택 범위 내의 셀들에 소정의 시각적 피드백을 적용하는 단계; 및

(f) 사용자가 접촉을 해제하는 시점에, 수평 이동 거리가 셀 폭의 소정 비율 이상이거나 수평 이동 속도가 소정의 속도 이상인 경우 작업 수행을 확정하고, 그렇지 않은 경우 원래 상태로 복귀시키는 단계;

를 포함하며,

상기 (a) 단계에 의해, 상기 (b)~(f) 단계에 의한 스와이프 작업의 수행 전후 어느 시점에서든 사용자가 임의의 셀을 탭하면 별도의 모드 해제 없이 해당 항목의 상세 뷰어가 즉시 실행되고, 상세 뷰어에서 복귀 후에도 별도의 모드 재진입 없이 스와이프 작업을 계속 수행할 수 있는 것을 특징으로 하는, 선택 모드 전환이 불필요한 스와이프 제스처를 이용한 항목 선택 및 작업 수행 방법.

## 9. 요약서

### 요약

본 발명은 터치스크린을 구비한 전자 장치에서 그리드 형태로 배열된 콘텐츠 항목에 대해, 별도의 선택 모드 전환 없이 스와이프 제스처를 이용하여 항목을 선택하고 소정의 작업을 수행하되, 상태 변경 이후에도 탭 입력에 의한 상세 열람이 항상 유지되는 방법 및 장치에 관한 것이다.

탭 제스처에는 항상 상세 뷰어 실행 역할을, 수평 스와이프 제스처에는 항상 항목 선택/작업 역할을 고정 할당하여, 스와이프에 의한 항목 상태 변경 전후를 포함한 어떠한 시점에서도 탭에 의한 상세 열람 기능이 유지된다. 그리드 셀 위에서 시작된 수평 스와이프 제스처의 진행도에 따라 셀 위에 오버레이를 점진적으로 표시하는 커튼 효과를 적용하고, 스와이프 진행 중 접촉점이 대상 셀을 벗어나 인접 셀에 도달하면 자동으로 다중 항목 선택 모드로 전환한다. 접촉 해제 시 거리 또는 속도 조건에 따라 작업 수행을 확정한다.

이에 의해, 사용자는 스와이프 작업 수행 직후에도 다른 항목을 탭하여 상세 뷰어에 즉시 진입할 수 있고, 복귀 후 별도의 모드 재진입 없이 스와이프 작업을 이어갈 수 있으며, 한 번의 연속된 스와이프 제스처로 단일 및 다중 항목에 대한 선택과 작업 수행이 가능하다.

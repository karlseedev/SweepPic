### logStats() 빈 출력 패턴
- `ImagePipeline.logStats()`, `MemoryThumbnailCache.logStats()` — 통계 계산만 하고 **아무것도 출력하지 않음**. 출력을 추가하거나 메서드 자체 제거

### CaseIterable 불필요 채택
- `QualityResult`, `QualitySignal`, `SessionStatus`, `PreviewCandidate` 등 — `.allCases` 호출 없이 채택

### SimilarPhotoGroup nonisolated 남발
- `struct`에 불필요한 `nonisolated` 키워드 12곳

### PhotoPageViewController deinit 누락
- `fullSizeRequestCancellable` cancel이 deinit에서 누락됨

### RenderABTest 초기값
- `activeTest = .glassEffectOff` — 테스트 상태 잔존, `.baseline`으로 복원 필요

### SimilarityConstants var → let
- `personMatchThreshold`, `greyZoneThreshold` — 값 변경 없으므로 `let`으로 변경 가능
